🚜 Misión 5: El Registro de la Granja

¡La granja está creciendo! Han llegado nuevos animales y necesitamos un sistema para registrarlos a todos de forma organizada. Tu misión es dar vida al formulario de registro usando JavaScript.

🛠 Preparación​

  1. Descarga de recursos: Descarga los archivos necesarios desde el siguiente enlace:
    🔗 Descargar Recursos Actividad 5 Formularios
  2. Desarrollo: Abre la carpeta con VS Code. Debes escribir todos los comandos de JavaScript en el archivo script.js.

📋 Lista de Tareas​

1️⃣ Control de Acceso​

Necesitas tener el control total del formulario.

  • Misión: Selecciona el formulario con id #registro-animal y guárdalo en una constante llamada formulario.
  • Pista: Usa document.getElementById().

2️⃣ Intercepción de Datos​

No queremos que la página se recargue al pulsar el botón.

  • Misión: Añade un escuchador del evento submit al formulario. Dentro, usa preventDefault() para evitar que se envíe de forma tradicional.
  • Pista: evento.preventDefault().

3️⃣ Validación en Vivo (Nombre)​

Queremos avisar al granjero si el nombre del animal es demasiado corto mientras escribe.

  • Misión: Selecciona el input nombre. Añade un evento input que compruebe si la longitud del texto es menor a 3 caracteres. Si es así, muestra un mensaje en el div #mensaje-error que diga "¡El nombre es demasiado corto!". Si es correcto, limpia el mensaje.
  • Pista: Usa .value.length y .textContent.

4️⃣ Vista Previa Dinámica​

El granjero quiere ver cómo queda la ficha antes de guardarla.

  • Misión: Haz que, al cambiar el Tipo de Animal o el Estado de Salud, se actualicen automáticamente los campos correspondientes en la sección de "Vista Previa" (#prev-nombre, #prev-tipo y #prev-salud).
  • Pista: Escucha el evento change o input en los elementos correspondientes.

5️⃣ Contador de Caracteres​

Es importante saber cuánto espacio ocupan las observaciones.

  • Misión: En el campo observaciones, añade un evento input. Cada vez que el usuario escriba, actualiza el texto de #contador-caracteres con el número actual de caracteres.
  • Pista: Accede a la propiedad .value.length del textarea.

6️⃣ El Nombre Prohibido​

No permitas que se usen nombres inapropiados como "malo" o "feo".

  • Misión: Si el nombre incluye alguna de estas palabras prohibidas (puedes usar .toLowerCase().includes()), muestra un mensaje de error y cambia el borde del input a color rojo.
  • Pista: Usa classList.add() o style.borderColor.

7️⃣ Campos Condicionales​

¡Solo las vacas producen leche!

  • Misión: Si el Tipo de Animal seleccionado es "Vaca", muestra el div #grupo-produccion usando style.display = 'block'. Si no es una vaca, ocúltalo.
  • Pista: Escucha el evento change en el select del tipo.

8️⃣ Semáforo de Salud​

Queremos un indicador visual rápido de la salud del animal.

  • Misión: Si el estado de salud es "Enfermo", cambia el color de fondo de la tarjeta de vista previa (#preview-card) a un tono rojizo suave. Si es "Saludable", ponlo en un tono verdoso o blanco.
  • Pista: Modifica style.backgroundColor.

9️⃣ Reset del Registro​

Necesitamos una forma rápida de limpiar todo el sistema.

  • Misión: Haz que al pulsar el botón #boton-reset, el formulario se limpie completamente (puedes usar .reset()) y todos los campos de la vista previa vuelvan a mostrar un guion ("-").
  • Pista: Escucha el evento click en el botón de reset.

🔟 El Gran Registro (Submit)​

Cuando todo esté listo, muestra los datos finales.

  • Misión: En el evento submit, si el formulario es válido, construye un objeto con todos los datos y muestra una alerta con un resumen: "Registrado: [Nombre] ([Tipo]) de [Edad] meses".
  • Pista: Puedes usar new FormData(formulario) para recoger los datos fácilmente.

🏆 Objetivo Final​

Al finalizar, habrás logrado:

  • Manipular un formulario complejo y sus eventos.
  • Validar datos y filtrar contenido.
  • Crear interfaces dinámicas que responden a las acciones del usuario.
  • Gestionar estados visuales mediante estilos y clases.

¡Tu granja ahora es la más avanzada tecnológicamente del valle! 🐄🚀
