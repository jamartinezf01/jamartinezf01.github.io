3.7.7 Interacción con Formularios

Los formularios son una parte esencial de las páginas web, ya que permiten a los usuarios interactuar con la aplicación enviando datos. JavaScript ofrece una amplia gama de propiedades y métodos para acceder, validar y manejar eventos relacionados con los formularios. Este apartado explica cómo interactuar de manera efectiva con formularios mediante el DOM. Las propiedades y eventos más comunes son...

Propiedad/MétodoDescripción
document.formsDevuelve una colección de todos los formularios del documento.
form.elementsDevuelve todos los elementos dentro de un formulario específico.
valueDevuelve o establece el valor de un campo de entrada.
checkedIndica si un checkbox o radio está seleccionado.
checkValidity()Comprueba si un campo es válido según las restricciones definidas.
setCustomValidity()Define un mensaje personalizado para errores de validación.
Eventos
submitSe dispara al enviar un formulario.
changeSe dispara al cambiar y perder el foco de un campo.
inputSe dispara cada vez que el usuario modifica el contenido de un campo.

Acceso a Formularios y Elementos​

1. document.forms​

  • Descripción:

    • document.forms es una colección que contiene todos los formularios (<form>) presentes en el documento.
    • Puedes acceder a un formulario por su índice o por el valor de su atributo name o id.

  • Ejemplo Básico:

<form id="formulario1" name="miFormulario">
  <input type="text" name="usuario" placeholder="Usuario">
  <input type="password" name="contraseña" placeholder="Contraseña">
</form>
<script>
  const formulario = document.forms['miFormulario']; // Acceso por nombre
  console.log(formulario.id); // "formulario1"
</script>

2. form.elements​

  • Descripción:

    • form.elements devuelve una colección de todos los elementos de formulario dentro de un formulario específico (campos de entrada, botones, etc.).
    • Los elementos se pueden acceder por su índice o por el valor del atributo name.

  • Ejemplo Básico:

<form id="formulario">
  <input type="text" name="usuario" placeholder="Usuario">
  <input type="password" name="contraseña" placeholder="Contraseña">
</form>
<script>
  const formulario = document.getElementById('formulario');
  const usuario = formulario.elements['usuario'];
  const contraseña = formulario.elements['contraseña'];
  console.log(usuario.placeholder); // "Usuario"
  console.log(contraseña.type);     // "password"
</script>

Validación de Formularios​

JavaScript permite validar formularios antes de enviarlos, proporcionando una mejor experiencia de usuario y reduciendo la carga del servidor.

1. Propiedades Comunes​

value​

  • Devuelve o establece el valor de un campo de entrada.
console.log(usuario.value); // Devuelve el texto introducido
usuario.value = "NuevoUsuario"; // Cambia el valor del campo

checked​

  • Indica si un checkbox o radio button está seleccionado.
const checkbox = formulario.elements['aceptar'];
console.log(checkbox.checked); // true o false
checkbox.checked = true; // Marca el checkbox

Ejemplo: Validar Campos Requeridos​

<form id="formulario">
  <input type="text" name="usuario" placeholder="Usuario" required>
  <button type="submit">Enviar</button>
</form>
<script>
  const formulario = document.getElementById('formulario');
  formulario.addEventListener('submit', (evento) => {
    const usuario = formulario.elements['usuario'];
    if (!usuario.value) {
      evento.preventDefault(); // Evita el envío
      alert('Por favor, completa el campo de usuario.');
    }
  });
</script>

2. Métodos de Validación​

checkValidity()​

  • Verifica si un elemento cumple con todas sus restricciones de validación (como required, minlength, pattern, etc.).
if (usuario.checkValidity()) {
  console.log('El campo es válido');
} else {
  console.log('El campo no es válido');
}

setCustomValidity()​

  • Establece un mensaje de error personalizado que aparece cuando el campo no es válido.
usuario.setCustomValidity('El nombre de usuario no puede estar vacío');

Ejemplo: Validar y Personalizar Mensajes​

<form id="formulario">
  <input type="email" name="correo" placeholder="Correo electrónico" required>
  <button type="submit">Enviar</button>
</form>
<script>
  const formulario = document.getElementById('formulario');
  formulario.addEventListener('submit', (evento) => {
    const correo = formulario.elements['correo'];
    if (!correo.checkValidity()) {
      evento.preventDefault();
      correo.setCustomValidity('Por favor, introduce un correo válido.');
      correo.reportValidity(); // Muestra el mensaje de validación
    }
  });
</script>

Manejo de Eventos de Formularios​

Los formularios generan eventos específicos que permiten interactuar y responder a las acciones del usuario.

1. Evento submit​

  • Se dispara cuando se intenta enviar un formulario.
  • Permite interceptar el envío para realizar validaciones.
formulario.addEventListener('submit', (evento) => {
  evento.preventDefault(); // Evita el envío
  console.log('Formulario enviado');
});

2. Evento change​

  • Se dispara cuando el valor de un campo cambia y pierde el foco.
usuario.addEventListener('change', () => {
  console.log(`Nuevo valor: ${usuario.value}`);
});

3. Evento input​

  • Se dispara cada vez que el usuario modifica el valor de un campo (sin esperar a perder el foco).
usuario.addEventListener('input', () => {
  console.log(`Valor actual: ${usuario.value}`);
});

Ejemplo de Manejo de Eventos​

<form id="formulario">
  <input type="text" name="usuario" placeholder="Usuario">
  <input type="password" name="contraseña" placeholder="Contraseña">
  <input type="checkbox" name="aceptar" id="aceptar">
  <label for="aceptar">Acepto los términos</label>
  <button type="submit">Enviar</button>
</form>
<script>
  const formulario = document.getElementById('formulario');

  formulario.addEventListener('submit', (evento) => {
    evento.preventDefault();
    const usuario = formulario.elements['usuario'];
    const contraseña = formulario.elements['contraseña'];
    const aceptar = formulario.elements['aceptar'];

    if (!usuario.value || !contraseña.value || !aceptar.checked) {
      alert('Por favor, completa todos los campos y acepta los términos.');
      return;
    }

    console.log('Formulario válido. Enviando...');
  });

  formulario.elements['usuario'].addEventListener('input', () => {
    console.log('El usuario está escribiendo...');
  });

  formulario.elements['aceptar'].addEventListener('change', (evento) => {
    console.log(`Checkbox: ${evento.target.checked ? 'Marcado' : 'Desmarcado'}`);
  });
</script>
