3.7.6 Navegación por el DOM

La navegación por el DOM permite acceder y recorrer los nodos de un documento para manipular su contenido o estructura. Los navegadores proporcionan varias propiedades que ayudan a moverse entre los nodos de un árbol DOM, permitiendo acceder a padres, hijos y hermanos de un nodo. Las propiedades más comunes son...

PropiedadDescripción
parentNodeDevuelve el nodo padre, sin importar el tipo.
childNodesDevuelve todos los nodos hijos (incluyendo texto y comentarios).
firstChild, lastChildDevuelve el primer/último nodo hijo, sin importar el tipo.
previousSibling, nextSiblingDevuelve el nodo hermano anterior/siguiente, sin importar el tipo.
parentElementDevuelve el nodo padre, pero solo si es un elemento.
childrenDevuelve solo los nodos hijos que son elementos.
firstElementChild, lastElementChildDevuelve el primer/último hijo que es un elemento.
previousElementSibling, nextElementSiblingDevuelve el nodo hermano anterior/siguiente que es un elemento.

Propiedades de Navegación​

1. parentNode​

  • Descripción:

    • Devuelve el nodo padre del nodo actual.
    • Si el nodo no tiene padre (por ejemplo, el nodo raíz document), devuelve null.

  • Ejemplo Básico:

<div id="contenedor">
  <p id="parrafo">Hola, DOM</p>
</div>
<script>
  const parrafo = document.getElementById('parrafo');
  console.log(parrafo.parentNode); // Nodo <div id="contenedor">
</script>

2. childNodes​

  • Descripción:

    • Devuelve una lista (NodeList) de todos los nodos hijos, incluidos nodos de texto, elementos y comentarios.
    • Los nodos de texto (espacios en blanco) se incluyen en la lista.

  • Ejemplo Básico:

<div id="contenedor">
  <p>Hola</p>
  <!-- Comentario -->
</div>
<script>
  const contenedor = document.getElementById('contenedor');
  console.log(contenedor.childNodes); // NodeList [<p>, #comment, #text]
</script>

3. firstChild y lastChild​

  • Descripción:

    • firstChild: Devuelve el primer nodo hijo, incluyendo nodos de texto o comentarios.
    • lastChild: Devuelve el último nodo hijo, incluyendo nodos de texto o comentarios.

  • Ejemplo Básico:

<div id="contenedor">
  <p>Hola</p>
  <!-- Comentario -->
</div>
<script>
  const contenedor = document.getElementById('contenedor');
  console.log(contenedor.firstChild); // Nodo <p>
  console.log(contenedor.lastChild);  // Nodo #comment
</script>

4. previousSibling y nextSibling​

  • Descripción:

    • previousSibling: Devuelve el nodo hermano anterior al nodo actual.
    • nextSibling: Devuelve el nodo hermano siguiente al nodo actual.
    • Incluyen nodos de texto, comentarios, etc.

  • Ejemplo Básico:

<div>
  <p id="parrafo1">Párrafo 1</p>
  <!-- Comentario -->
  <p id="parrafo2">Párrafo 2</p>
</div>
<script>
  const parrafo2 = document.getElementById('parrafo2');
  console.log(parrafo2.previousSibling); // Nodo #comment
  console.log(parrafo2.nextSibling);     // Nodo #text (espacio en blanco)
</script>

Propiedades Específicas de Elementos​

Las propiedades específicas de elementos (parentElement, children, firstElementChild, etc.) ignoran los nodos que no son elementos (como texto o comentarios), proporcionando una navegación más limpia.

1. parentElement​

  • Descripción:

    • Similar a parentNode, pero devuelve el nodo padre solo si es un nodo de tipo elemento. Si el nodo padre no es un elemento, devuelve null.

  • Ejemplo:

<div id="contenedor">
  <p id="parrafo">Hola, DOM</p>
</div>
<script>
  const parrafo = document.getElementById('parrafo');
  console.log(parrafo.parentElement); // Nodo <div id="contenedor">
</script>

2. children​

  • Descripción:

    • Devuelve una colección (HTMLCollection) de los nodos hijos que son elementos, excluyendo nodos de texto y comentarios.

  • Ejemplo:

<div id="contenedor">
  <p>Párrafo 1</p>
  <p>Párrafo 2</p>
</div>
<script>
  const contenedor = document.getElementById('contenedor');
  console.log(contenedor.children); // HTMLCollection [<p>, <p>]
</script>

3 firstElementChild y lastElementChild​

  • Descripción:

    • firstElementChild: Devuelve el primer nodo hijo que es un elemento.
    • lastElementChild: Devuelve el último nodo hijo que es un elemento.

  • Ejemplo:

<div id="contenedor">
  <p>Primero</p>
  <p>Último</p>
</div>
<script>
  const contenedor = document.getElementById('contenedor');
  console.log(contenedor.firstElementChild); // Nodo <p>Primero</p>
  console.log(contenedor.lastElementChild);  // Nodo <p>Último</p>
</script>

4. previousElementSibling y nextElementSibling​

  • Descripción:

    • previousElementSibling: Devuelve el nodo hermano anterior que es un elemento.
    • nextElementSibling: Devuelve el nodo hermano siguiente que es un elemento.

  • Ejemplo:

<div>
  <p id="parrafo1">Párrafo 1</p>
  <p id="parrafo2">Párrafo 2</p>
</div>
<script>
  const parrafo2 = document.getElementById('parrafo2');
  console.log(parrafo2.previousElementSibling); // Nodo <p id="parrafo1">
  console.log(parrafo2.nextElementSibling);     // null (no hay más hermanos)
</script>

Ejemplo​

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Navegación por el DOM</title>
</head>
<body>
  <div id="contenedor">
    <h1>Título</h1>
    <p id="parrafo1">Párrafo 1</p>
    <p id="parrafo2">Párrafo 2</p>
    <!-- Comentario -->
  </div>
  <script>
    const contenedor = document.getElementById('contenedor');
    const parrafo1 = document.getElementById('parrafo1');

    // Propiedades generales
    console.log(parrafo1.parentNode); // Nodo <div>
    console.log(contenedor.childNodes); // Incluye nodos de texto y comentarios
    console.log(contenedor.firstChild); // Nodo <h1>

    // Propiedades específicas de elementos
    console.log(contenedor.children); // HTMLCollection [<h1>, <p>, <p>]
    console.log(parrafo1.nextElementSibling); // Nodo <p id="parrafo2">

    // Navegación combinada
    console.log(contenedor.firstElementChild.nextElementSibling); // Nodo <p id="parrafo1">
  </script>
</body>
</html>
