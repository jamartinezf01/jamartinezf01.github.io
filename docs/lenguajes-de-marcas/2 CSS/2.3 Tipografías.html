2.3 Tipografías en CSS

La tipografía define cómo se lee y cómo se percibe el contenido. En esta sección verás las propiedades básicas, la decoración del texto, cómo importar fuentes (Google Fonts y @font-face) y cómo usar iconos editables de Google (Material Symbols).

1️⃣ Propiedades básicas de tipografía​

Estas propiedades controlan el tipo de letra (familia), el tamaño, el interlineado y el grosor del texto. Son la base para una lectura cómoda y coherente en toda la web.

🟩 font-family (familias y fallbacks)​

font-family define una fuente principal y alternativas (fallbacks) para asegurar que el texto se vea correctamente incluso si falla la principal.

PropiedadQué haceEjemplo recomendado
font-familyLista de fuentes con fallback"Inter", Arial, sans-serif
Ejemplo de aplicación de una pila de fuentes a todo el documento
body {
  font-family: "Inter", Arial, sans-serif;
}
en este ejemplo

En caso de que no se encuentre la funte "Iter" se aplicará la fuente "Arial" y posteriormente la "sans-serif"

🟨 Tamaño e interlineado​

Podemos ajusta el tamaño del texto con la propiedad font-size y la separación entre líneas para facilitar una lectura fluida con line-height.

PropiedadQué haceRecomendaciónEjemplo
font-sizeEstablece el tamaño de la fuenteBase ≈ 16px (o 1rem)font-size: 16px;
line-heightEstablece el espacio entre líneasSin unidad (unitless)line-height: 1.5;
Ejemplo de aplicación de diferentes tamaños e interlineados para encabezados y párrafos
h1 {
  font-size: 32px;   /* Título destacado */
  line-height: 1.2;
}
p {
  font-size: 16px;   /* Texto legible */
  line-height: 1.5;  /* Interlineado cómodo */
}
unidades de medida para tipografia

Si debes elegir una sola estrategia para tipografía, elige rem: mantiene la consistencia, respeta la accesibilidad y te permite escalar toda la interfaz ajustando solo el tamaño raíz. Reserva px para detalles finos donde el valor absoluto es realmente necesario.

🟧 Peso, estilo y espaciados​

Usa el grosor y la cursiva para crear jerarquía; ajusta espaciados en casos concretos (mayúsculas, titulares).

PropiedadQué haceValores útilesUso frecuente
font-weightEstablece el grosor de la fuente400 (normal), 700 (bold)Jerarquía
font-styleEstablece el estilo de la fuentenormal, italicÉnfasis
letter-spacingCambia el espacio entre letras en una palabrap. ej. 0.02emMayúsculas
word-spacingCambia el espacio entre palabras en un párrafop. ej. 0.05emBloques largos

  • CSS
  • HTML
  • Representación
h2 {
  font-weight: 800;
}
em {
  font-style: italic;
}
.badge {
  letter-spacing: 0.40em;
}

<h2>Título en negrita</h2>
<p>Texto con <em>cursiva</em> para énfasis y <span class="badge">ETIQUETAS</span> en mayúsculas.</p>

2️⃣ Decoración, maquetación y transformación​

Mediante la decoración y transformación de texto controlas subrayados/tachados, el flujo del texto y su representación para mejorar claridad y usabilidad.

🟩 Subrayado y tachado​

Haz los enlaces claramente distinguibles y marca cambios o estados del texto.

PropiedadQué controlaEjemplo
text-decoration-lineunderline, line-throughtext-decoration-line: underline;
text-decoration-colorColor del subrayadotext-decoration-color: #0ea5e9;
text-decoration-thicknessGrosor de la líneatext-decoration-thickness: 2px;
text-underline-offsetDistancia entre texto y subrayadotext-underline-offset: 3px;
  • CSS
  • HTML
  • Representación
a.demo {
  color: #0ea5e9;
  text-decoration-line: underline;
  text-decoration-color: #0ea5e9;
  text-decoration-thickness: 2px;
  text-underline-offset: 3px;
}
.del { text-decoration: line-through; }
<a class="demo" href="#">Enlace claramente subrayado</a>
<p class="del">Este texto aparece como eliminado.</p>

🟨 Alineación y flujo​

Introducción. Evita desbordes y mejora el bloque de lectura con opciones de alineación y control de espacios.

PropiedadUso principalNota
text-alignleft, center, justifySe aplica al contenedor
white-spaceEspacios/saltosnormal, pre-wrap, nowrap
overflow-wrapPartición de palabras largasoverflow-wrap: break-word;
  • CSS
  • HTML
  • Representación
.center { text-align: center; }
.nowrap { white-space: nowrap; }
.break  { overflow-wrap: break-word; }
<p class="center">Este texto está centrado.</p>
<p class="nowrap">Este texto no se partirá automáticamente en varias líneas.</p>
<p class="break">UnaPalabraExtraLargaQueDeOtraManeraSeDesbordaríaEnPantallasPequeñas</p>

🟧 Transformación de texto​

La propiedad text-transform no cambia el contenido, solo cómo se muestra en pantalla. Sirve para poner el texto en mayúsculas, minúsculas o capitalizar palabras sin modificar el HTML original.

ValorQué hacePara qué sirveEjemplo CSS
noneSin transformaciónMantener el texto tal cualtext-transform: none;
uppercaseConvierte a MAYÚSCULASBotones, etiquetas, acrónimostext-transform: uppercase;
lowercaseConvierte a minúsculasNormalizar entradas de usuariotext-transform: lowercase;
capitalizePrimera letra de cada palabra en mayúsculaTitulares cortos, nombres propiostext-transform: capitalize;
Importante

La transformación es visual. El DOM mantiene el texto original (útil para búsquedas, SEO y lectores de pantalla).

  • CSS
  • HTML
  • Representación
.demo {
  margin: 8px 0;
  padding: 6px 8px;
  border: 1px solid #e5e7eb;
  border-radius: 8px;
  background: #fff;
}

.u-upper {
  text-transform: uppercase;
}
.u-lower {
  text-transform: lowercase;
}
.u-capi {
  text-transform: capitalize;
}
.u-none {
  text-transform: none;
}
<p class="demo u-none">Texto original: “aprende css con ejemplos visibles”.</p>
<p class="demo u-upper">Texto a mayúsculas: “aprende css con ejemplos visibles”.</p>
<p class="demo u-lower">Texto a minúsculas: “APRENDE CSS CON EJEMPLOS VISIBLES”.</p>
<p class="demo u-capi">Capitalizar: “aprende css con ejemplos visibles”.</p>

3️⃣ Importar fuentes​

En la actualidad, es muy común utilizar Google Fonts como repositorio proveedor de tipografías para utilizar en nuestros sitios web.

En la propia página de Google Fonts podemos seleccionar las fuentes con las características deseadas y generar un código HTML con la tipografía (o colección de tipografías) que vamos a utilizar.

Para poder emplear las fuentes de google fonts deberemos seguir los siguientes pasos...

  1. Visita la página de Google Fonts
  2. Selecciona la fuente deseada (puedes usar más de una)
  3. Pulsa en Get Font
  4. Pulsa en Get Embed Code
  5. Copia e Inserta el <link> de la familia en el head de tu HTML y después aplica font-family con fallbacks.
  • HTML (link)
  • CSS (uso)
OJO esto va en el <head></head>
<!-- Inter 100/900 -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Onest:wght@100..900&display=swap" rel="stylesheet">
body { font-family: "Inter", Arial, sans-serif; }
h1   { font-weight: 700; }
p    { font-weight: 400; }
Consejo

display=swap en el enlace evita el “texto invisible” mientras carga la fuente remota.

4️⃣ Fuentes autoalojadas​

En ocasiones el uso de Google fonts hace que nuestras páginas se carguen con lentitud, o no contiene la fuente que nosotros queremos. En ese caso, también podemos emplear fuentes que el usuario no tenga instaladas en el sistema suministrándosela nosotros mismos. Para ello deberemos tener el archivo de fuente en el servidor en un formato estándar (woff, woff2, ttf, eot…).

Para ello emplearemos la regla CSS @font-face. Coloca los archivos en tu proyecto y declara cada peso/estilo.

  • Estructura
  • @font-face
/assets/fonts/
  brand-regular.woff2
  brand-bold.woff2
@font-face {
  font-family: "Brand";
  src: url("/assets/fonts/brand-regular.woff2") format("woff2");
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}
@font-face {
  font-family: "Brand";
  src: url("/assets/fonts/brand-bold.woff2") format("woff2");
  font-weight: 700;
  font-style: normal;
  font-display: swap;
}

body { font-family: "Brand", Arial, sans-serif; }
Licencias

Comprueba que la licencia de la fuente permite su autohospedaje y distribución.

5️⃣ Iconos editables de Google​

También podemos usar Google Fonts como fuente de iconos. Para ello deberemos enlazar la hoja de estilos de iconos Google en nuestra cabecera HTML.

Los Material Symbols son iconos como fuente: se escalan bien y puedes editar relleno, peso y tamaño con CSS.

Para poder usar los iconos, importa la familia y usa el nombre del icono como texto.

  • HTML (link)
  • CSS (clase)
  • HTML (icono)
  • Representación
<link href="https://fonts.googleapis.com/css2?family=Material+Symbols+Rounded:FILL@0..1&display=swap" rel="stylesheet">
.material-symbols-rounded{
  font-family: "Material Symbols Rounded";
  font-weight: normal; font-style: normal; line-height: 1;
  display: inline-block; vertical-align: middle;
  font-size: 24px; color: currentColor;
  font-variation-settings: "FILL" 1, "wght" 400, "GRAD" 0, "opsz" 24;
}
<span class="material-symbols-rounded" aria-hidden="true">home</span>
<span class="sr-only">Inicio</span>

Accesibilidad

Si el icono aporta significado, añade texto visible u oculto. Si es decorativo, marca aria-hidden="true".

✅ Buenas prácticas​

buenas prácticas
  • font-size: 16px y line-height: 1.5 como base cómoda.
  • Define fallbacks en font-family.
  • En Google Fonts, pide solo los pesos que necesitas.
  • text-underline-offset y grosor adecuado para subrayados claros.
  • Prefiere Material Symbols o SVG a imágenes raster para iconos.
