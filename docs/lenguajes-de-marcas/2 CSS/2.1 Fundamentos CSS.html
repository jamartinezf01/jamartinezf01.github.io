2.1 Fundamentos de CSS

En este punto se presentan los conceptos base para escribir reglas CSS correctas y predecibles: sintaxis, unidades de medida y formatos de color Incluye ejemplos prácticos y fragmentos listos para copiar.

fuentes

Todas la imagenes de esta sección están extraidas de la página WEB "MANZ.DEV"

1️⃣ Sintaxis básica​

La sintaxis CSS se basa en 4 elementos principales, los documentos CSS, las reglas CSS y Los selectores CSS

🟩 Reglas CSS​

Una regla CSS está compuesta por un selector y un bloque de declaraciones:

  • Selector: indica a qué elementos HTML se aplica el estilo.
  • Propiedad: qué aspecto se quiere modificar (ej. color, margin).
  • Valor: el ajuste concreto (ej. red, 16px).
  • ; : cada declaración termina en punto y coma.
  • comentarios Las reglas también pueden contener comentarios. Un comentario es un fragmento de texto entre /* y */ con anotaciones o aclaraciones para el desarrollador (el navegador los ignorará).
Ejemplo de regla básica
/* Estilo predeterminado para el documento */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}
errores frecuentes
  • Olvidar el ; al final de cada declaración.
  • Escribir propiedad= valor en lugar de propiedad: valor.
  • Usar nombres de clase con espacios (class="mi clase") → no es valido: usa guion (mi-clase) o camelCase.

🟨 Documentos CSS​

Los documentos CSS (hoja de estilos) se forman por cientos de reglas de este estilo, con diferentes selectores, propiedades y valores. Habitualmente, cada bloque declarado suele incluir múltiples reglas .

Ejemplo de documento CSS
/* Estilo predeterminado para el documento */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

/* Estilo para cabeceras */
h1 {
    color: #143b63ff;
    text-align: center;
}
h2 {
    color: #6597c9ff;
}

/* Estilo para listas */
ol, ul {
    margin-left: 20px;
}

🟧 Selectores Básicos​

Un selector CSS es una referencia a uno o varios elementos HTML, escrito generalmente desde código CSS. Aunque el esquema completo es mucho más amplio, para empezar, podemos centrarnos en los 3 tipos básicos de selectores que son los de etiqueta, id y clases:

Ejemplos
  • Selector por etiqueta
  • Selector por ID
  • Selector por Clase
Todas las etiquetas div del documento HTML tendrán el fondo rojo
div {
  background-color: red;
}

Cuando una etiqueta HTML tiene un atributo id específico se puede seleccionar ese elemento único mediante el nombre de su id. Se hace añadiendo un almohadilla # al inicio del selector.

Solo la etiquetas con id principal tendrá el fondo rojo
#principal {
  background-color: red;
}

Selecciona todas las etiquetas de un documento HTML que tengan un atributo class con el mismo valor. Se hace añadiendo un punto . al inicio del selector.

Solo la etiquetas con class='fondo-rojo' tendrá el fondo rojo
.fondo-rojo {
  background-color: red;
}
selectores múltiples y combinados

Podemos emplear múltiples selectores para aplicar la misma regla a diferentes elementos separándolas mediante una coma ,:

Asignamos la fuente Arial a todos los tipos de encabezado
h1, h2, h3, h4, h5, h6 {
  font-family: Arial;
}

También podemos combinar selectores para ser mucho más precisos a la hora de seleccionar los elementos a los que se aplicarán las reglas.

  • Ejemplo 1
  • Ejemplo 2
Aplica el estilo solo a los párrafos dentro de un elemento de clase contenedor.
.contenedor p {
  font-size: 16px;
  color: green;
}
Aplica el estilo solo a los elementos de la clase especifico dentro de un elemento con clase contenedor.
.contenedor .especifico {
  color: blue;
  font-weight: bold;
}

2️⃣ Propiedades​

Una propiedad CSS es el nombre estandarizado que indica qué aspecto visual de un elemento HTML se quiere controlar.

🟩 Propiedades básicas​

Como vimos en la anterior unidad, las propiedades básicas CSS son las siguientes.

PropiedadDescripciónEjemplo
colorDefine el color del texto.p { color: red; }
background-colorEstablece el color de fondo de un elemento.body { background-color: lightblue; }
font-sizeCambia el tamaño de la fuente.h1 { font-size: 32px; }
font-familyDefine la tipografía.p { font-family: Arial, sans-serif; }
text-alignAlinea el texto (left, right, center, justify).h1 { text-align: center; }
font-weightDefine el grosor del texto (normal, bold).strong { font-weight: bold; }
marginEstablece el espacio exterior de un elemento.p { margin: 20px; }
paddingEstablece el espacio interior (relleno) de un elemento.div { padding: 10px; }
borderAñade un borde alrededor de un elemento.p { border: 1px solid black; }
width / heightDefinen el ancho y alto de un elemento.img { width: 200px; height: auto; }

🟨 Otras propiedades​

La tabla del punto anterior no recoge una parte minúscula de las propiedades CSS, Existe una infinidad de propiedades que iremos utilizando a lo largo de esta unidad.

En el siguiente enlace de la página de "MANZ.DEV" encontraras una guía completa de propiedades CSS organizadas por categorías.

🟧 Custom properties​

Las CSS Custom Properties son variables nativas de CSS. Se definen con --nombre y se usan con var(--nombre). El objetivo principal es evitar escribir múltiples veces un mismo valor, y en su lugar, ponerle un nombre más lógico, semántico y fácil de recordar, que hará referencia al valor real. De esta forma será mucho más legible y más fácil de mantener.

  • Definición: pares --token: valor; asociados a un alcance (scope) en el árbol CSS.
  • Uso: propiedad: var(--token, fallback); — el segundo parámetro (fallback) es opcional y actúa como valor de respaldo.
  • Ventajas: theming (claro/oscuro), diseño consistente (espacios/colores), cambios en tiempo real, menos repetición.
Ejemplo de uso de Custom Properties
:root {
  --color-principal: hsl(200 90% 45%);
  --radio: 12px;
}

.boton {
  background: var(--color-principal);
  border-radius: var(--radio);
}
:root

:root es una pseudoclase (ya veremos más adelante que es esto) que selecciona el elemento raiz del documento. En documentos HTML, :root coincide con <html> pero es más específica por lo que se recomienda su uso. Se usa, sobre todo, para definir estilos “globales” y CSS Custom Properties (variables) que heredara el resto de la pagina..

3️⃣ Unidades CSS​

Las unidades sirven para indicar tamaños y distancias en CSS (anchos, altos, margenes, fuente, etc.). Elegir bien la unidad mejora la responsividad, la legibilidad y evita sorpresas de maquetación. Principalmente hay dos tipos:

  • Absolutas - Son un tipo de medida fija que no cambia, que no depende de ningún otro factor.
  • Relativas - Son aquellas que fijan su tamaño dependiendo de algún otro valor.

🟩 Unidades básicas​

Con saber 6 unidades ya puedes maquetar la mayoria de interfaces.

UnidadQue significaTipoCuándo usarla
pxPixel CSS (tamaño fijo)AbsolutaBordes finos, iconos raster, sombras.
%Porcentaje del contenedorRelativaAnchos fluidos (porciones de columna).
emRelativa al tamaño de fuente del elementoRelativaEspaciados que crecen con el componente.
remRelativa al tamaño de fuente raiz (html)RelativaTipografia y espaciado consistente en toda la pagina.
vw1% del ancho de la ventanaRelativaTitulos/anchos fluidos.
vh1% de la altura de la ventanaRelativaSecciones “pantalla”.
  • px
  • %
  • em
  • rem
  • vw y vh
Unidad fija
.avatar { border: 2px solid #333; }  /* 2px siempre */
porcentaje del contenedor
.foto { width: 50%; } /* ocupa la mitad del ancho del padre */
relativa al propio elemento
.boton {
  font-size: 16px;
  padding: 0.5em 1em; /* crece si sube font-size del boton */
}
relativa al raiz
html { font-size: 16px; }   /* base */
h1 { font-size: 2rem; }     /* 32px si la base es 16 */
p  { font-size: 1rem; }     /* 16px */
relativa al tamaño de ventana
```css
.hero {
  min-height: 100vh; /* ocupa toda la altura visible */
}
.titulo {
  font-size: 5vw; /* crece/disminuye con el ancho de la ventana */
}

🟨 ¿Qué unidad elegir?​

tip
  • Texto y espaciado general → rem
  • Anchos fluidos dentro de un contenedor → %
  • Bordes y detalles finos → px
  • Componente que escala con su propia fuente → em
  • Bloques a pantalla completa o titulos fluidos → vh / vw

🟧 Errores frecuentes​

cuidado
  • Usar px para toda la tipografia → dificulta la accesibilidad (mejor rem).
  • Poner height: % sin que el padre tenga altura definida → no funcionara.
  • Usar solo vh en moviles puede producir “saltos” al aparecer barras del navegador.

🟥 Ejemplo​

  • HTML
  • CSS
index.html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8" />
  <title>Unidades CSS (basico)</title>
  <link rel="stylesheet" href="css/estilos.css" />
</head>
<body>
  <header class="hero">
    <h1 class="titulo">Mi sitio</h1>
  </header>

  <main class="contenedor">
    <img class="foto" src="foto.jpg" alt="Ejemplo" />
    <button class="boton">Comprar</button>
  </main>
</body>
</html>
css/estilos.css
html { font-size: 16px; }
body { font-family: system-ui, sans-serif; }

.hero { min-height: 60vh; display: grid; place-items: center; }
.titulo { font-size: clamp(1.5rem, 4vw, 3rem); } /* rem + vw (opcional) */

.contenedor { width: 90%; max-width: 60rem; margin: 2rem auto; }

.foto { width: 50%; border: 2px solid #333; } /* % + px */
.boton {
  font-size: 1rem;          /* 16px */
  padding: 0.5em 1em;       /* escala con el boton */
  border-radius: 8px;
}

4️⃣ Colores​

Los colores en CSS se usan para texto, fondos, bordes, sombras, gradientes, etc. Las formas más comunes de definir color son: nombres, hexadecimal, RGB(a) y HSL(a). También hay palabras clave útiles como transparent, currentColor e inherit.

🟩 Formatos principales​

FormatoEjemploCuándo usarlo
Nombrecolor: red;Rápido para pruebas. Conjunto limitado.
Hex#1f2937, #fff, #1f2937ccMuy usado; admite alfa con 8 dígitos (#RRGGBBAA).
RGB(a)rgb(31 41 55), rgba(31, 41, 55, .8)Familiar, preciso; con alfa para transparencias.
HSL(a)hsl(220 13% 18%), hsla(220, 13%, 18%, .8)Intuitivo para aclarar/oscurecer (luminosidad).
Palabras clavetransparent, currentColor, inheritCasos especiales útiles (ver abajo).
  • Nombre
  • Código Hexadecimal
  • RGB y RGBA
  • HSL y HSLA
Usando el nombre del color en Inglés
h1 { color: royalblue; }
Se usa el código Hexadecimal del colos (de 6 a 8 dígitos)
.card {
  background: #ffffff;   /* 6 dígitos */
  border-color: #e5e7eb; /* gris claro */
}
.badge {
  background: #0ea5e980; /* HEX con alfa (~50% opacidad) */
}
usa la combinación de los colores Red / Green / Blue
.alert {
  color: rgb(153 27 27);          /* sintaxis con espacios también es válida */
  border: 1px solid rgb(220 38 38);
  background: rgba(220, 38, 38, 0.12); /* con alfa (comas también válido) */
}
Usa una combinación de Tono / %Saturación / %Lumninosidad 
.btn {
  background: hsl(200 90% 45%);   /* tono, saturación y luminosidad */
  color: hsl(0 0% 100%);          /* blanco */
}
.btn:hover {
  background: hsla(200, 90%, 45%, .9); /* con alfa (comas también válido) */
}

🟨 Transparencia (canal alfa)​

Casi todos los formatos tienen variante con alfa (transparencia):

  • HEX: #RRGGBBAA
  • rgba(r, g, b, a)
  • hsla(h, s, l, a)

Cuanto menor el valor de a (0→1), más transparente.

🟧 Palabras clave útiles​

  • transparent: color totalmente transparente (equivale a rgba(0,0,0,0)).

  • currentColor: usa el mismo color que color del elemento.

    .icono {
      color: #111;                  /* color del texto */
      border: 2px solid currentColor; /* reutiliza el mismo color */
    }

  • inherit: hereda el color del elemento padre.

🟥 Recomendaciones​

tip
  • Usa HEX, RGB(a) o HSL(a) según te resulte más cómodo; HSL facilita aclarar/oscurecer.
  • transparent, currentColor e inherit resuelven casos comunes sin repetir valores.
  • Centraliza tu paleta con Custom Properties en :root y podrás cambiar temas fácilmente.

🟪 Ejemplo​

  • HTML
  • CSS
index.html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8" />
  <title>Colores en CSS</title>
  <link rel="stylesheet" href="css/colores.css" />
</head>
<body>
  <main class="demo">
    <h1>Colores en CSS</h1>
    <p class="texto">Texto con HSL</p>
    <button class="btn">Accion</button>
    <span class="badge">Badge</span>
  </main>
</body>
</html>
css/colores.css
/* Variables globales */
:root {
  --fg: hsl(222 47% 11%);
  --bg: #f8fafc;
  --accent: hsl(200 90% 45%);
}

body { color: var(--fg); background: var(--bg); font: 1rem/1.6 system-ui, sans-serif; }
.demo { width: min(90vw, 60rem); margin: 2rem auto; }

.texto { color: hsl(220 13% 18%); }

.btn {
  background: var(--accent);
  color: #fff;
  border: 1px solid rgb(14 165 233);
  padding: .625rem 1rem;
  border-radius: 10px;
  cursor: pointer;
}
.btn:hover { background: hsla(200, 90%, 45%, .9); }

.badge {
  color: #111;
  border: 2px solid currentColor; /* igual que el color del texto */
  background: #0ea5e980;          /* HEX con alfa */
  padding: .25rem .5rem;
  border-radius: 999px;
}
