2.8.2 Flexbox

El modelo Flexbox (Flexible Box Layout) es uno de los sistemas de diseño más versátiles de CSS. Fue creado para resolver los problemas de maquetación que aparecían con los floats y el positioning, ofreciendo una manera mucho más intuitiva de alinear, distribuir y adaptar elementos en una o varias filas o columnas.

Con Flexbox, el navegador reparte el espacio de manera automática y flexible, permitiendo crear estructuras adaptables con un código sencillo. Esto lo convierte en una de las herramientas más importantes para diseñar interfaces modernas y layouts responsive.

1️⃣ Concepto general y estructura básica​

El sistema Flexbox se compone de dos tipos de elementos:

  • El contenedor flexible, activado mediante display: flex.
  • Los ítems flexibles, que son los elementos hijos directos dentro del contenedor.

El contenedor se comporta como una caja flexible en la que los elementos hijos pueden expandirse, contraerse, alinearse y distribuirse a lo largo de un eje principal y un eje secundario.

Ejes en Flexbox:

  • Eje principal (main axis): define la dirección en la que se distribuyen los ítems (horizontal por defecto).
  • Eje secundario (cross axis): perpendicular al principal, controla la alineación vertical u horizontal según el caso.

El layout Flexbox nos permite estructurar y ordenar elementos de un modo muy flexible. Nos permite alinearlos, ordenarlos, cambiar su dirección , su tamaño...

Para realizar todo esto contamos con propiedades que nos permiten cambiar las características del contenedor de elementos y de los elemntos propiamente dichos.

2️⃣ Propiedades del contenedor​

🟩 Activación display: flex​

Para activar Flexbox basta con aplicar la propiedad display: flex a un contenedor. A partir de ese momento, todos los hijos directos se convierten en ítems flexibles.

<div class="contenedor">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>
.contenedor {
  display: flex;
  background: #e6f0ff;
  padding: 10px;
}
.item {
  background: #99b3ff;
  padding: 20px;
  margin: 5px;
  border-radius: 8px;
}
en este ejemplo

Los tres elementos se distribuyen horizontalmente en una fila. Si el espacio no alcanza, se reducirán proporcionalmente según las reglas de Flexbox.

🟨 Dirección flex-direction​

La dirección en la que se disponen los ítems se controla con la propiedad flex-direction.

ValorDescripciónFlujo visual
rowDistribuye los ítems en una fila, de izquierda a derecha.→
row-reverseFila, pero en orden inverso.←
columnColumna de arriba a abajo.↓
column-reverseColumna, pero de abajo a arriba.↑
.contenedor {
  display: flex;
  flex-direction: column;
  background: #eaf0ff;
  height: 250px;
}
en este ejemplo

Los ítems se apilan verticalmente dentro del contenedor. Cambiando el valor de flex-direction se controla si el flujo es horizontal o vertical.

🟧 Distribución justify-content​

La propiedad justify-content controla cómo se alinean y distribuyen los ítems a lo largo del eje principal (horizontal por defecto).

🔹 Alinear elementos​

ValorEfecto visualDescripción
flex-startÍtems alineados al inicio.Valor por defecto.
centerCentra los ítems en el eje principal.Distribución uniforme centrada.
flex-endÍtems alineados al final.Aparecen al borde derecho o inferior.

🔹 Distribuir elementos​

ValorEfecto visualDescripción
space-betweenEspacio igual entre ítems.Sin espacio en los extremos.
space-aroundEspacio igual alrededor de cada ítem.Márgenes laterales iguales.
space-evenlyEspacio igual entre y en los extremos.Totalmente equilibrado.

🔹 Agrupar elementos​

Los contenedores flexibles solo saben cómo colocar elementos que tienen un nivel de profundidad (es decir, sus elementos secundarios). No les importa en lo más mínimo lo que hay dentro de sus elementos flexibles.  Esto es útil para agrupar elementos.

<div class="contenedor">
  <div class="item">1</div>
  <div class="grupo">
     <div class="item">2</div>
     <div class="item">3</div>
  </div>
</div>
.contenedor {
  display: flex;
  justify-content: space-between;
  background: #e6f0ff;
  padding: 10px;
}
.item {
  background: #99b3ff;
  padding: 20px;
  margin: 5px;
  border-radius: 8px;
}
en este ejemplo

al agrupar los elementos 2 y 3 hacemos que el contenedor solo identifique dos elementos con un nivel de profundidad y reparta el espacio solo entre estos dos

🟥 Alineación align-items​

La propiedad align-items define cómo se alinean los ítems en el eje secundario (perpendicular al principal).

ValorDescripción
flex-startAlinea los ítems al inicio del eje secundario.
centerCentra los ítems verticalmente (si el eje principal es horizontal).
flex-endLos alinea al final del eje secundario.
stretchAjusta la altura de los ítems para ocupar todo el espacio disponible.
baselineAlinea según la línea base del texto.
Eje principal y eje secundario

Debemos tener cuidado con el uso de las propiedades justify-content y align-items. Generalmente entendemos que una es para la alineación horizontal y otra para la alineación vertical pero esto no es así. Si nos fijamos estamos hablando de alineación en el eje primario y secundario. Estos ejes son por defecto horizontal y vertical pero si cambiamos el flex-direction estos cambian por lo que el uso de las propiedades justify-content y align-items tambien cambia

🟪 Desbordamiento flex-wrap​

En ocasiones la cantidad de lementos sobrepasa la capacidad del contenedor flexible, para manejar estas situaciones contamos con la propiedad flex-wrap. Mediante esta propiedad podremos elegir si los elementos deben desbordar el contenedor o, si por el contrario, el contenedor debe organizar los elementos de manera automática.

  • flex-wrap: nowrap
  • flex-wrap: wrap

🟦 Separación entre ítems gap​

La propiedad gap (antes llamada grid-gap) permite definir el espacio entre los ítems flexibles sin tener que usar márgenes manualmente. Aporta un código más limpio y evita problemas de cálculo cuando los ítems se envuelven (flex-wrap).

ValorDescripción
gap: 10px;Añade 10 píxeles entre cada ítem.
row-gap y column-gapPermiten controlar espacios horizontal y vertical por separado.
HTML
 <div class="contenedor">
    <img class='caja uno' src='imagenes/uno.png'/> 
    <img class='caja dos' src='imagenes/dos.png'/> 
    <img class='caja tres' src='imagenes/tres.png'/>            
    <img class='caja cuatro' src='imagenes/cuatro.png'/> 
    <img class='caja cinco' src='imagenes/cinco.png'/>    
 </div>
CSS
  .contenedor {
    display: flex;
    flex-wrap: nowrap;
    justify-content: center;
    align-items: center;
    height: 300px; width: 60%;
    margin: 0 auto;
    border: 3px solid #1100ff;
    gap: 20px;
  }
  .caja{
    display: block;
    border: 1px solid #fff;
    background-size: cover;
    width: 100px;
  }
Ventajas de usar gap
  • No afecta a los bordes del contenedor.
  • Es más fácil de mantener que los márgenes.
  • Funciona también con Grid Layout.

3️⃣ Propiedades de los ítems​

Hasta ahora hemos visto que aplicando algunas propiedades CSS al contenedor flexible (display: flex) podemos organizar automáticamente su contenido, controlar su dirección o ajustar su distribución.

Sin embargo, Flexbox también permite modificar el comportamiento individual de los elementos dentro del contenedor. Estas propiedades se aplican a los ítems flexibles (hijos directos) y permiten personalizar su alineación, tamaño, orden o separación.

🟩 Autoalineación align-self​

La propiedad align-self permite alinear un elemento concreto de forma independiente, ignorando la alineación general establecida por align-items en el contenedor.

Esto es muy útil cuando un ítem necesita una posición distinta del resto en el eje secundario (vertical, si la dirección es row).

ValorDescripción
autoUsa la alineación del contenedor (align-items).
flex-startAlinea el ítem al inicio del eje secundario.
flex-endLo alinea al final del eje secundario.
centerCentra verticalmente el ítem dentro del contenedor.
stretchAjusta su tamaño para ocupar todo el alto disponible.
 <div class="contenedor">
    <img class='caja uno' src='imagenes/uno.png'/> 
    <img class='caja dos' src='imagenes/dos.png'/> 
    <img class='caja tres' src='imagenes/tres.png'/>            
    <img class='caja cuatro' src='imagenes/cuatro.png'/> 
    <img class='caja cinco' src='imagenes/cinco.png'/>    
 </div>
.contenedor {
    display: flex;
    flex-wrap: nowrap;
    justify-content: center;
    align-items: center;
    height: 300px; width: 50%;
    margin: 0 auto;
    border: 3px solid #1100ff;
  }
  .caja{
    display: block;
    border: 1px solid #fff;
    background-size: cover;
    width: 100px;
  }
  .dos{
    align-self: flex-start;
  }
  .cuatro{
    align-self: flex-end;
  }

🟨 Tamaño flexible – flex​

La propiedad flex es la más importante para definir el tamaño y comportamiento dinámico de los ítems dentro de un contenedor Flexbox. Controla simultáneamente cómo crecen (flex-grow), se encogen (flex-shrink) y cuál es su tamaño base (flex-basis).

flex: <grow> <shrink> <basis>;

Por ejemplo:

flex: 1 1 200px;
en este ejemplo
  • Puede crecer (1),
  • Puede reducirse (1),
  • Su tamaño base es 200px.
EjemploEquivale aResultado visual
flex: 1;flex: 1 1 0;Todos los elementos comparten el espacio disponible de forma igual.
flex: 2;flex: 2 1 0;Crece el doble que los elementos con flex: 1.
flex: 0 0 150px;Tamaño fijo de 150px, sin crecimiento ni reducción.
  • Si los elementos de un contenedor no tienen la propiedad flex tendrán su tamaño original.
  • Si el valor de la propiedad flex es igual en todos los elementos, compartiran el tamaño total del contenedor.
  • Si el valor de la propiedad flex es mayor en un os elementos que en otros, estos tendran un tamaño proporcionalmente mayor que los otros.

  • FLEX IGUAL
  • FLEX DIFERENTE

Además, podemos mezclar y combinar cajas flexibles con cajas de ancho fijo. flex: initial recurre a la propiedad width explícita del elemento. Esto nos permite combinar cajas estáticas y flexibles de formas complejas.

🟧 Orden – order​

Por defecto, los ítems de un contenedor Flexbox se muestran en el mismo orden en que aparecen en el HTML. Con la propiedad order, se puede modificar el orden visual sin alterar el código fuente.

Esto resulta útil cuando se quiere adaptar el diseño para distintas resoluciones o cambiar la disposición visual en pantallas pequeñas.

ValorDescripción
0Valor por defecto (el orden del HTML).
1, 2, 3, etc.Muestra los ítems según su número. Cuanto menor sea el valor, antes se mostrará.
Valores negativosPermiten mostrar el ítem antes que los demás.

4️⃣ Ajustar el footer con flex​

En ocasiones, cuando el contenido de la página no es suficiente para llenar la pantalla vemos que debajo de nuestro footer queda espacio en blanco. Para evitar esto deberemos ajustar el footer al pie de la ventana. Existen varias formas de hacer esto, una de ellas es usando display flex.

Haremos que las etiquetas <html> y <body> sean contenedores flex en dirección vertical que he ocupen todo el alto de la pantalla y posteriormente daremos al contenido principal de la página la propiedad flex:1 ocupando todo el contenedor posible.

📊 Resumen​

Propiedad CSSFunción / Uso principal
display: flexSe utiliza para crear un contenedor flexible que activa el modelo Flexbox.
flex-directionSe utiliza si se necesitan columnas en lugar de filas.
row-reverse / column-reverseSe utilizan para invertir el orden de los elementos en filas o columnas.
justify-contentSe utiliza para definir la alineación horizontal de los elementos dentro del contenedor.
align-itemsSe utiliza para definir la alineación vertical de los elementos en el eje secundario.
orderSe utiliza para personalizar el orden de los elementos individuales.
align-selfSe utiliza para alinear verticalmente elementos individuales, anulando align-items.
flexSe utiliza para crear cajas flexibles que pueden estirarse o encogerse según el espacio.
orderSe utiliza para personalizar el orden de los elementos individuales.
gapSe utiliza para indicar la separación entre elementos, en vez de margin.
