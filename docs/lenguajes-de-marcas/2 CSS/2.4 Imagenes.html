2.4 Imágenes de fondo

Es posible que, en lugar de utilizar un color de fondo quieras utilizar imágenes. Las imágenes de fondo permiten decorar elementos HTML añadiendo una imagen detrás de su contenido. Se aplican principalmente mediante la propiedad background-image y se pueden combinar con colores, gradientes y múltiples fondos.

1️⃣ Añadir una imagen de fondo​

La propiedad principal es:

background-image: url("ruta/imagen.jpg");

Se puede aplicar a cualquier elemento, aunque los más comunes son el body, contenedores <div> o secciones.

  • CSS
  • HTML
  • Representación
    body {
      background-image: url("img/Pikachu.png");
    }
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Fondo simple</title>
  <link rel="stylesheet" href="estilos.css">
</head>
<body>
  <h1>Ejemplo de fondo</h1>
  <p>Este texto se muestra sobre una imagen de fondo.</p>
</body>
</html>

2️⃣ Propiedades del fondo​

Una vez establecida una imagen de fondo con background-image, se puede personalizar la forma en la que se mostrará dicha imagen mediante propiedades como las siguientes.

PropiedadDescripciónEjemplo
background-colorColor de fondo (puede combinarse con imagen).background-color: #f0f0f0; Gradientes
background-imageRuta de la imagen.background-image: url("fondo.jpg");
background-repeatControla la repetición: repeat, repeat-x, repeat-y, no-repeat.background-repeat: no-repeat;
background-positionPosición inicial de la imagen: left, right, center, top, bottom o coordenadas (50% 50%).background-position: center;
background-sizeTamaño de la imagen: auto, cover, contain, o valores fijos.background-size: cover;
background-attachmentDefine si el fondo se desplaza con el contenido (scroll) o se fija (fixed).background-attachment: fixed;
background-originDefine el área de origen del fondo (padding-box, border-box, content-box).background-origin: border-box;
background-clipDefine hasta dónde se muestra el fondo.background-clip: padding-box;
background-blend-mode

En caso de que la imagen de fondo interfiera con el contenido podremos aclararla mezclándola con algun color mediante la propiedad background-blend-mode

  • CSS
  • Representación
    body {
      background-color: rgba(255, 255, 255, 0.8);  
      background-image: url("img/Pikachu.png");
      background-blend-mode: lighten;
    }

🟩 Propiedad abreviada background​

Se pueden combinar todas las propiedades en una sola línea:

background: url("paisaje.jpg") no-repeat center/cover fixed;

➡ Equivale a:

background-image: url("paisaje.jpg");
background-repeat: no-repeat;
background-position: center;
background-size: cover;
background-attachment: fixed;

3️⃣ Fondos múltiples​

CSS permite aplicar varias imágenes de fondo a un mismo elemento. Cada imagen se separa con una coma y las capas se apilan de arriba hacia abajo (la primera es la más visible).

  • CSS
  • HTML
  • Representación
    .imagen {
        width: 100px;
        height: 100px;
        background-image: 
            url("/img/pikachu.png"),
            url("/img/fondo.jpg");
        background-repeat: no-repeat, repeat;
        background-position: top right, center;
        background-size: 100px, cover;
    }
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Fondo simple</title>
  <link rel="stylesheet" href="estilos.css">
</head>
<body>
  <div class="imagen"></div>
</body>
</html>
en este ejemplo
  • El primer fondo (pikachu) se coloca arriba a la derecha y no se repite.
  • El segundo (principal) ocupa toda la pantalla (cover).

✅ Buenas prácticas​

buenas prácticas
  • Usa imágenes de fondo ligeras y optimizadas (formato .jpg o .webp).
  • Emplea background-size: cover; para adaptarlas a distintas resoluciones.
  • Evita textos sobre imágenes con poco contraste; puedes usar un degradado o capa semitransparente.
  • Combina imágenes y colores de fondo para mejorar la legibilidad y el rendimiento.
