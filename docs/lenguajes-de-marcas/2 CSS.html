UT2 - CSS

CSS (Cascading Style Sheets) es un lenguaje que permite definir la presentación y estilo de los documentos HTML. Gracias a CSS, se controla la apariencia de un sitio web: colores, tipografía, tamaños, márgenes, disposición de los elementos, animaciones, etc.

Es uno de los 3 pilares del diseño WEB:

  • HTML define la estructura del contenido.
  • CSS define la apariencia visual.
  • JavaScript define el comportamiento dinámico.

1️⃣ ¿Por qué usar CSS?​

  • Separa el contenido (HTML) de la presentación (CSS).
  • Permite reutilizar estilos en varias páginas.
  • Facilita el mantenimiento de sitios web.
  • Mejora la accesibilidad y la experiencia de usuario.

2️⃣ Cómo se aplica CSS​

Existen tres formas principales de aplicar CSS en una página web:

🟩 CSS en línea (inline)​

Se escribe directamente en el atributo style de una etiqueta HTML.

<p style="color: red; font-size: 20px;">
  Este texto es rojo y tiene tamaño 20px.
</p>
poco recomendada

Se recomienda solo para pruebas rápidas o estilos muy puntuales.

🟨 CSS interno (dentro del HTML)​

Se coloca dentro de la etiqueta <style> en el archivo HTML.

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo CSS Interno</title>
  <style>
    p {
      color: blue;
      font-size: 18px;
    }
  </style>
</head>
<body>
  <p>Este texto es azul y tiene tamaño 18px.</p>
</body>
</html>
solo casos puntuales

Se recomienda solo para pruebas rápidas o estilos muy puntuales.

➡️ Útil para páginas pequeñas o prototipos.

🟧 CSS externo (archivo separado)​

Se crea un archivo .css y se enlaza con <link> en el HTML.

  • Código HTML
  • Estilo CSS
  • Representación
index.html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo CSS Externo</title>
  <link rel="stylesheet" href="css/estilos.css">
</head>
<body>
  <p>Este texto está estilizado desde un archivo externo.</p>
</body>
</html>
estilos.css
p {
  color: green;
  font-size: 22px;
}
recomendada

Es la forma recomendada porque permite reutilizar y mantener mejor el código.

🟥 Orden de aplicación​

Prioridad de aplicación (de menor a mayor prioridad)

  1. Estilos del navegador (User Agent).
  2. Estilos del usuario (hoja personalizada del usuario).
  3. Autor – externo (<link rel="stylesheet" href="...">).
  4. Autor – <style> interno (en el propio HTML).
  5. Autor – inline (style="..." en el elemento) → suele imponerse por su alta especificidad.

Entre externo y <style> interno, si el selector y la especificidad empatan, gana el que aparece más tarde (orden de aparición en el documento). El inline pertenece al autor, pero tiene especificidad muy alta y normalmente ganará a 3 y 4 si no hay !important.

3️⃣ Primer selector básico​

Los selectores indican qué elementos HTML se van a estilizar. Ejemplo: aplicar estilo a todos los párrafos <p>.

Ejempo de selector básico
p {
  color: purple;
  font-weight: bold;
}
info

Este código hace que todos los párrafos de la página se vean en morado y en negrita.
