6.1 Dar estilo a XML con CSS

1. ¿Cómo se vincula el XML con el CSS?​

Para aplicar un archivo .css a un documento XML, se incluye una línea al principio del XML, justo después de la declaración XML:

<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/css" href="estilo.css"?>

El navegador usará esa hoja de estilos para mostrar el XML con apariencia personalizada.

2. Limitaciones de CSS aplicado a XML​

✅ Lo que SÍ puede hacer CSS en XML:

  • Cambiar colores, bordes, márgenes, fuentes.
  • Añadir iconos con content.
  • Estilizar etiquetas visibles.

❌ Lo que NO se puede hacer con CSS en XML

  • Acceder a atributos de elementos vacíos.
  • Reordenar datos o filtrar contenido.
  • Crear nuevas estructuras (tablas, listas).
  • Aplicar condiciones o lógica de presentación.
  • Generar distintos formatos de salida.

3. Estilos CSS más utilizados para aplicar a XML​

Cuando se aplica CSS directamente sobre XML (sin transformarlo a HTML), lo que se puede hacer es dar estilo a los nombres de los elementos (etiquetas), porque no hay clases ni IDs como en HTML. Es más limitado, pero útil para una visualización básica en el navegador.

Los estilos más útiles y comunes son:

🔸 Selección de elementos por nombre​

maxima {
  color: red;
  font-weight: bold;
  font-size: 18px;
}

Se aplica un estilo visual destacado a la temperatura máxima, resaltándola en rojo y con mayor tamaño.

🔸 Márgenes, espaciado y bordes​

dia {
  display: block;
  margin-bottom: 15px;
  padding: 10px;
  border: 1px solid #ccc;
  border-radius: 8px;
  background-color: #eaf2f8;
}

Cada dia se presenta como un bloque separado con bordes suaves y fondo azul claro, facilitando la lectura de la predicción diaria.

🔸 Colores y fuentes​

minima {
  color: blue;
  font-family: Arial, sans-serif;
}

Se muestra la temperatura mínima en azul con una fuente clara.

🔸 Alineación y fondo decorativo​

velocidad {
  font-style: italic;
  text-align: right;
  background-color: #fff3cd;
  padding: 5px;
}

La velocidad del viento aparece alineada a la derecha, en cursiva y con un fondo amarillo claro que la resalta visualmente.

Propiedad CSSDescripciónEjemplo valores
colorDefine el color del texto.black, blue, #333, red
font-weightEstablece el grosor del texto (por ejemplo, negrita).normal, bold, lighter
font-sizeAjusta el tamaño de la fuente.12px, 1em, small, large
displayDefine cómo se muestra un elemento.block, inline, inline-block,none
marginAñade espacio exterior alrededor del elemento. También se puede usar por lados (margin-top, margin-left, etc.)0, 10px 15px, auto
paddingAñade espacio interno (relleno). También se puede usar por lados (padding-top, etc.)5px, 10px 15px
font-familyEspecifica la fuente del texto.Arial, Verdana, serif
font-styleEstablece si el texto es normal o cursiva.normal, italic
text-alignAlinea el contenido horizontalmente.left, center, right
background-colorDefine el color de fondo del elemento.#fff, #f0f0f0, yellow
borderAñade un borde alrededor del elemento. También puede usarse border-top, border-left, etc.1px solid black, none
RECORDAR
  • No se puede aplicar CSS a atributos (@atributo), solo a elementos.
  • Es una solución estética y temporal, útil para visualizar datos XML de forma más clara, pero no transforma el documento.
