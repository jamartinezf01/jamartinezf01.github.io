6.3 XSLT

1. ¿Qué es XSLT?​

XSLT (eXtensible Stylesheet Language Transformations) es un lenguaje diseñado para transformar documentos XML en otros formatos, como HTML, texto plano o incluso nuevos documentos XML.

Este lenguaje forma parte de XSL (eXtensible Stylesheet Language), que también incluye:

  • XPath, para localizar y seleccionar nodos dentro del XML.
  • XSL-FO, utilizado para formatear documentos impresos (no se tratará en esta unidad).

2. ¿Para qué se usa?​

El objetivo principal de XSLT es separar el contenido (datos) de su presentación (forma de visualizarlos), permitiendo transformar automáticamente los datos originales en:

  • Páginas web HTML a partir de datos XML.
  • Informes o listados en texto plano.
  • Nuevas estructuras XML adaptadas a otros sistemas.

3. ¿Cómo funciona?​

El proceso de transformación sigue un esquema como el siguiente:

  1. Se parte de un documento XML con datos.
  2. Se aplica una hoja de estilo XSLT que define cómo deben transformarse esos datos.
  3. Se genera un nuevo documento (HTML, XML, texto...).

4. ¿Por qué es útil?​

XSLT es una herramienta muy versátil cuando se necesita mostrar, adaptar o exportar información almacenada en XML. Su capacidad para transformar datos en diferentes formatos permite reutilizar el mismo contenido en múltiples contextos sin duplicar esfuerzo ni modificar el archivo original.

Beneficios principales:​

  • Permite mostrar los datos XML en forma de página web (HTML).
  • Automatiza tareas de conversión de datos.
  • Facilita la reutilización de los datos con diferentes presentaciones.
  • Se integra fácilmente en entornos web o sistemas externos.
Ejemplo de uso

Un archivo XML que contiene productos de una tienda puede transformarse mediante XSLT para generar:

  • Una página web HTML que muestre el catálogo a los usuarios.
  • Un listado en texto plano para su impresión.
  • Un nuevo XML con distinta estructura, adaptado a un sistema externo de gestión de inventario.
<item>
  <code>p01</code>
  <description>Polvos de cuerno de unicornio</description>
  <type>Ingredientes</type>
  <unitPrice>25</unitPrice>
  <stock>10</stock>
</item>
