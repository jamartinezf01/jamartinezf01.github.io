3.7.3 Manipulación de Elementos

La manipulación de elementos es una de las tareas más comunes en JavaScript al trabajar con el DOM. Nos permite cambiar el contenido, atributos, clases y estilos de los elementos, lo que resulta fundamental para crear aplicaciones dinámicas e interactivas.

1. Cambiar Contenido​

innerHTML​

  • Descripción:

    • Permite establecer o recuperar el contenido HTML de un elemento.
    • Convierte el texto introducido en código HTML, lo que puede incluir etiquetas y estructuras.

  • Ejemplo:

<div id="miDiv">Contenido original</div>
<script>
  const elemento = document.getElementById('miDiv');
  console.log(elemento.innerHTML); // "Contenido original"

  // Cambiar contenido con etiquetas
  elemento.innerHTML = "<p>Nuevo contenido con <strong>HTML</strong>.</p>";
</script>
peligro

Usar innerHTML con contenido dinámico puede ser un riesgo de seguridad si no se valida adecuadamente, ya que puede introducir ataques XSS. Los ataques XSS (Cross-Site Scripting) son un tipo de vulnerabilidad de seguridad que permite a un atacante inyectar scripts maliciosos en una página web que otros usuarios pueden visitar.

textContent​

  • Descripción:

    • Permite establecer o recuperar solo el contenido de texto del elemento.
    • No interpreta etiquetas HTML, simplemente las muestra como texto literal.

  • Ejemplo:

<div id="miDiv">Contenido <strong>original</strong></div>
<script>
  const elemento = document.getElementById('miDiv');
  console.log(elemento.textContent); // "Contenido original"

  // Cambiar contenido como texto
  elemento.textContent = "Este es solo texto.";
</script>
  • Diferencias entre innerHTML y textContent:
PropiedadInterpreta HTMLMuestra Solo Texto
innerHTMLSíNo
textContentNoSí

2. Modificar Atributos​

getAttribute()​

  • Descripción:

    • Recupera el valor de un atributo específico de un elemento.

  • Ejemplo:

<img id="miImagen" src="imagen.jpg" alt="Descripción">
<script>
  const imagen = document.getElementById('miImagen');
  console.log(imagen.getAttribute('src')); // "imagen.jpg"
  console.log(imagen.getAttribute('alt')); // "Descripción"
</script>

setAttribute()​

  • Descripción:

    • Establece el valor de un atributo para un elemento. Si el atributo no existe, lo crea.

  • Ejemplo:

<img id="miImagen" src="imagen.jpg">
<script>
  const imagen = document.getElementById('miImagen');
  // Cambiar atributo existente
  imagen.setAttribute('src', 'nueva-imagen.jpg');
  console.log(imagen.getAttribute('src')); // "nueva-imagen.jpg"

  // Crear un nuevo atributo
  imagen.setAttribute('title', 'Nueva imagen');
  console.log(imagen.getAttribute('title')); // "Nueva imagen"
</script>

Propiedades como id, className, etc.​

  • También se pueden manipular atributos directamente mediante propiedades específicas como id, className, src, href, etc.

  • Ejemplo:

<a id="miEnlace" href="https://example.com" class="enlace">Ir a Example</a>
<script>
  const enlace = document.getElementById('miEnlace');
  console.log(enlace.href); // "https://example.com"

  // Cambiar id y clase
  enlace.id = "nuevoId";
  enlace.className = "nuevaClase";
</script>

3. Añadir y Eliminar Clases​

Las clases de los elementos se gestionan de manera eficiente mediante la propiedad classList.

classList.add()​

  • Añade una o más clases al elemento.
const elemento = document.getElementById('miElemento');
elemento.classList.add('nuevaClase');

classList.remove()​

  • Elimina una o más clases del elemento.
const elemento = document.getElementById('miElemento');
elemento.classList.remove('nuevaClase');

classList.toggle()​

  • Añade una clase si no existe y la elimina si ya está presente.
const elemento = document.getElementById('miElemento');
elemento.classList.toggle('otraClase'); // Añade "otraClase" si no está, la elimina si está

4. Cambiar Estilos en Línea​

Los estilos CSS de un elemento se pueden manipular directamente a través de la propiedad style. Esto solo afecta a los estilos en línea y no altera las reglas CSS definidas en archivos externos o <style>.

  • Sintaxis:
elemento.style.propiedadCSS = "valor";
  • Ejemplo:
<div id="miDiv" style="color: blue;">Texto con estilo inicial</div>
<script>
  const elemento = document.getElementById('miDiv');

  // Cambiar el color del texto
  elemento.style.color = "red";

  // Cambiar el fondo
  elemento.style.backgroundColor = "yellow";

  // Cambiar tamaño de fuente
  elemento.style.fontSize = "20px";
</script>

Propiedades CSS Especiales​

OJO
  • Las propiedades CSS que contienen guiones (-) como background-color se escriben en camelCase (backgroundColor).

Ejemplo Completo​

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Manipulación de Elementos</title>
  <style>
    .resaltado {
      color: white;
      background-color: green;
    }
  </style>
</head>
<body>
  <div id="miDiv" class="miClase">Texto inicial</div>
  <button id="botonCambiar">Cambiar</button>

  <script>
    const div = document.getElementById('miDiv');
    const boton = document.getElementById('botonCambiar');

    boton.addEventListener('click', () => {
      // Cambiar contenido
      div.textContent = "Texto cambiado";

      // Modificar atributo
      div.setAttribute('title', 'Nuevo título');

      // Añadir clase
      div.classList.add('resaltado');

      // Cambiar estilo
      div.style.border = "2px solid black";
    });
  </script>
</body>
</html>
