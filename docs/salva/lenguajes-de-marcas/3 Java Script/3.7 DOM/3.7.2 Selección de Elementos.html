3.7.2 Selección de Elementos

La selección de elementos es uno de los pasos fundamentales para manipular el DOM en JavaScript. Existen diferentes métodos para localizar y acceder a los elementos del árbol DOM, según los identificadores, clases, etiquetas u otros selectores.

Métodos Clásicos para Seleccionar Elementos​

Los métodos clásicos para buscar elementos en el DOM son fáciles de usar pero carecen de la flexibilidad de los más modernos. Si necesitas localizar un elemento específico, lo más recomendable es emplear el método getElementById() para buscarlo mediante su atributo id. Para otros casos, puedes optar por cualquiera de los otros tres métodos, los cuales siempre devuelven una colección de elementos:

1. getElementById()​

Este método selecciona un elemento basado en el valor único del atributo id.

  • Sintaxis:
document.getElementById('id');

  • Características:

    • Solo selecciona un único elemento.
    • Es uno de los métodos más rápidos para acceder a un elemento.
    • Devuelve null si no encuentra el elemento.

  • Ejemplo:

<div id="miDiv">Contenido del div</div>
<script>
  const elemento = document.getElementById('miDiv');
  console.log(elemento.textContent); // "Contenido del div"
</script>

2. getElementsByClassName()​

Selecciona todos los elementos que tienen un nombre de clase específico. Devuelve una colección dinámica de nodos (HTMLCollection).

  • Sintaxis:
document.getElementsByClassName('clase');

  • Características:

    • Devuelve una colección de nodos (HTMLCollection).
    • Es sensible al nombre exacto de la clase (no selecciona clases parciales).
    • La colección se actualiza automáticamente si cambian los elementos en el DOM.

  • Ejemplo:

<div class="miClase">Elemento 1</div>
<div class="miClase">Elemento 2</div>
<script>
  const elementos = document.getElementsByClassName('miClase');
  console.log(elementos.length); // 2
  console.log(elementos[0].textContent); // "Elemento 1"
</script>

3. getElementsByTagName()​

Selecciona todos los elementos que coinciden con un nombre de etiqueta específico.

  • Sintaxis:
document.getElementsByTagName('tag');

  • Características:

    • Devuelve una colección dinámica de nodos (HTMLCollection).
    • Se puede usar tanto en el document como en un nodo específico.
    • Útil para seleccionar etiquetas genéricas como div, p, img, etc.

  • Ejemplo:

<p>Primer párrafo</p>
<p>Segundo párrafo</p>
<script>
  const parrafos = document.getElementsByTagName('p');
  console.log(parrafos.length); // 2
  console.log(parrafos[1].textContent); // "Segundo párrafo"
</script>

4 getElementsByName()​

  • Descripción:

    • Este método selecciona todos los elementos que tienen un atributo name con el valor especificado.
    • Devuelve una colección de nodos (NodeList) con todos los elementos que coincidan.

  • Usos comunes:

    • Frecuentemente utilizado para trabajar con formularios donde los campos tienen el atributo name.

  • Sintaxis:

document.getElementsByName('name');
  • Ejemplo:
<form>
  <input type="text" name="usuario" value="Juan">
  <input type="password" name="usuario" value="12345">
</form>
<script>
  const elementos = document.getElementsByName('usuario');
  console.log(elementos.length); // 2
  console.log(elementos[0].value); // "Juan"
</script>

Métodos Modernos para Seleccionar Elementos​

Los métodos modernos querySelector() y querySelectorAll() han revolucionado la forma de buscar elementos en el DOM gracias a su flexibilidad y compatibilidad con los selectores CSS. A diferencia de los métodos tradicionales, permiten realizar búsquedas más avanzadas, como seleccionar elementos mediante combinaciones de clases, atributos específicos o incluso pseudoclases. Esto los convierte en herramientas poderosas y versátiles para trabajar con estructuras HTML complejas y para adaptarse a los requisitos de diseño y funcionalidad de aplicaciones web modernas.

1. querySelector()​

  • Descripción:

    • Devuelve el primer elemento que coincide con el selector CSS especificado.
    • Si no encuentra ningún elemento, devuelve null.

  • Características:

    • Permite seleccionar elementos por:
      • ID (#id).
      • Clase (.clase).
      • Etiqueta (etiqueta).
      • Selectores compuestos (..., .clase > etiqueta, #id.etiqueta).
      • Pseudoclases (:hover, :first-child, etc.).
    • Es más lento que métodos como getElementById para selecciones simples, pero más flexible.

  • Sintaxis:

document.querySelector('selectorCSS');
  • Ejemplo:
<div id="contenedor">
  <p class="texto">Primer párrafo</p>
  <p class="texto">Segundo párrafo</p>
</div>
<script>
  const elemento = document.querySelector('.texto');
  console.log(elemento.textContent); // "Primer párrafo"
</script>

2. querySelectorAll()​

  • Descripción:

    • Devuelve todos los elementos que coinciden con el selector CSS especificado.
    • El resultado es una lista estática (NodeList), que no se actualiza si cambian los elementos del DOM.

  • Características:

    • Permite usar los mismos selectores CSS que querySelector.
    • La lista devuelta puede ser recorrida con métodos de iteración como forEach.

  • Sintaxis:

document.querySelectorAll('selectorCSS');
  • Ejemplo:
<div id="contenedor">
  <p class="texto">Primer párrafo</p>
  <p class="texto">Segundo párrafo</p>
</div>
<script>
  const elementos = document.querySelectorAll('.texto');
  console.log(elementos.length); // 2

  elementos.forEach((el) => {
    console.log(el.textContent); // Imprime cada párrafo
  });
</script>

Ejemplos Avanzados de Selectores CSS​

1. Selección por ID​

const elemento = document.querySelector('#miId');

2. Selección por Clase​

const elementos = document.querySelectorAll('.miClase');

3. Selectores Compuestos​

// Selecciona el primer párrafo dentro de un div con ID "contenedor"
const elemento = document.querySelector('#contenedor p');

// Selecciona todos los elementos con clase "activo" dentro de un div
const elementos = document.querySelectorAll('div.activo');

4. Uso de Pseudoclases​

// Selecciona el primer hijo de un div
const primerHijo = document.querySelector('div:first-child');

// Selecciona todos los elementos impares de una lista
const elementosImpares = document.querySelectorAll('li:nth-child(odd)');

5. Selectores de Atributos​

// Selecciona elementos con atributo "data-role"
const elementos = document.querySelectorAll('[data-role]');

// Selecciona un input con tipo "text"
const inputTexto = document.querySelector('input[type="text"]');

Casos Prácticos​

1. Resaltar el Primer Elemento con Clase "destacado"​

<div class="destacado">Elemento 1</div>
<div class="destacado">Elemento 2</div>
<script>
  const primerDestacado = document.querySelector('.destacado');
  primerDestacado.style.backgroundColor = 'yellow';
</script>

2. Iterar sobre Todos los Párrafos​

<p>Texto 1</p>
<p>Texto 2</p>
<p>Texto 3</p>
<script>
  const parrafos = document.querySelectorAll('p');
  parrafos.forEach((parrafo) => {
    console.log(parrafo.textContent);
  });
</script>

Ventajas de querySelector y querySelectorAll​

  1. Flexibilidad:
    • Permiten usar cualquier selector CSS.
  2. Simplicidad:
    • Reducen la necesidad de múltiples métodos (getElementById, getElementsByClassName, etc.).
  3. Compatibilidad:
    • Amplia compatibilidad con navegadores modernos.

Diferencias entre Selecciones Estáticas y Dinámicas​

1. Colecciones Dinámicas (HTMLCollection):​

  • Métodos como getElementsByClassName() y getElementsByTagName() devuelven una colección dinámica.
  • Si el DOM cambia (se añaden o eliminan elementos), la colección se actualiza automáticamente.

Ejemplo:

const elementos = document.getElementsByTagName('div');
console.log(elementos.length); // Número inicial

// Crear un nuevo elemento
const nuevoDiv = document.createElement('div');
document.body.appendChild(nuevoDiv);

console.log(elementos.length); // La colección se actualiza automáticamente

2. Listas Estáticas (NodeList):​

  • Métodos como querySelectorAll() devuelven una lista estática.
  • Los cambios en el DOM no afectan a la lista ya creada.

Ejemplo:

const elementos = document.querySelectorAll('div');
console.log(elementos.length); // Número inicial

// Crear un nuevo elemento
const nuevoDiv = document.createElement('div');
document.body.appendChild(nuevoDiv);

console.log(elementos.length); // La lista no cambia, sigue siendo estática

Recomendaciones Prácticas​

sigue estas recomendaciones...

  1. Para seleccionar un único elemento por su ID:

    • Usar getElementById() por su rapidez.

  2. Para trabajar con múltiples elementos:

    • Usar querySelectorAll() para obtener una lista estática o getElementsByClassName() para una colección dinámica.

  3. Cuando se necesitan selectores avanzados:

    • Usar querySelector() y querySelectorAll() para aprovechar la potencia de los selectores CSS.
