3.4.3 Cadenas de Texto

Las cadenas de texto (strings) en JavaScript son uno de los tipos de datos más utilizados. Representan secuencias de caracteres y se pueden usar para almacenar y manipular texto. En este manual, exploraremos las operaciones más comunes, métodos útiles y buenas prácticas para trabajar con cadenas de texto. ✨

Definición de Cadenas de Texto​

En JavaScript, las cadenas de texto se definen usando:

  • Comillas dobles (") 🟦
  • Comillas simples (') 🟪
  • Comillas invertidas (`) (para template literals 📝)

🔍 Ejemplos:​

let saludo1 = "Hola, mundo";   // Comillas dobles
let saludo2 = 'Hola, mundo';   // Comillas simples
let saludo3 = `Hola, mundo`;   // Template literal

Propiedades de las Cadenas​

Longitud (length) 📏​

La propiedad length devuelve el número de caracteres en una cadena, incluyendo espacios y símbolos.

Ejemplo:

let texto = "JavaScript";
console.log(texto.length); // 10

Métodos Comunes para Manipular Cadenas​

JavaScript ofrece numerosos métodos para manipular cadenas. A continuación, veremos los más importantes: 🛠️

1. Transformar Texto 🔄​

🔠 toUpperCase​

Convierte todos los caracteres de una cadena a mayúsculas.

Ejemplo:

let texto = "hola";
console.log(texto.toUpperCase()); // "HOLA"

🔡 toLowerCase​

Convierte todos los caracteres de una cadena a minúsculas.

Ejemplo:

let texto = "HOLA";
console.log(texto.toLowerCase()); // "hola"

2. Concatenar Cadenas 🔗​

➕ Operador +​

Une dos o más cadenas.

Ejemplo:

let saludo = "Hola";
let nombre = "Juan";
console.log(saludo + " " + nombre); // "Hola Juan"

🧩 Método concat​

Otra forma de concatenar cadenas.

Ejemplo:

let saludo = "Hola";
let nombre = "Juan";
console.log(saludo.concat(" ", nombre)); // "Hola Juan"

💡 Template Literals​

Usando comillas invertidas (```), puedes interpolar variables directamente dentro de una cadena.

Ejemplo:

let saludo = "Hola";
let nombre = "Juan";
console.log(`${saludo}, ${nombre}`); // "Hola, Juan"

3. Acceder a Caracteres 🔎​

🖇️ Notación de Corchetes ([])​

Accede a un carácter en una posición específica.

Ejemplo:

let texto = "JavaScript";
console.log(texto[0]); // "J"
console.log(texto[4]); // "S"

🔍 Método charAt​

Similar a [], devuelve el carácter en una posición específica.

Ejemplo:

let texto = "JavaScript";
console.log(texto.charAt(0)); // "J"

4. Buscar Texto 🔍​

📍 indexOf​

Devuelve el índice de la primera aparición de una subcadena. Si no se encuentra, devuelve -1.

Ejemplo:

let texto = "Hola, mundo";
console.log(texto.indexOf("mundo")); // 6
console.log(texto.indexOf("JavaScript")); // -1

📌 lastIndexOf​

Devuelve el índice de la última aparición de una subcadena.

Ejemplo:

let texto = "Hola, mundo, hola";
console.log(texto.lastIndexOf("hola")); // 13

✔️ includes​

Devuelve true si una subcadena está presente.

Ejemplo:

let texto = "Hola, mundo";
console.log(texto.includes("mundo")); // true
console.log(texto.includes("JavaScript")); // false

5. Extraer Subcadenas ✂️​

🔪 slice​

Extrae una parte de la cadena desde un índice inicial hasta (pero no incluyendo) un índice final.

Ejemplo:

let texto = "JavaScript";
console.log(texto.slice(0, 4)); // "Java"
console.log(texto.slice(4));    // "Script"

🚫 substring​

Similar a slice, pero no acepta índices negativos.

Ejemplo:

let texto = "JavaScript";
console.log(texto.substring(0, 4)); // "Java"
console.log(texto.substring(4));    // "Script"

🛑 substr​

Extrae una parte de la cadena basada en un índice inicial y una longitud.

Ejemplo:

let texto = "JavaScript";
console.log(texto.substr(4, 6)); // "Script"

6. Reemplazar Texto 🔄​

✏️ replace​

Reemplaza la primera aparición de una subcadena con otra.

Ejemplo:

let texto = "Hola, mundo";
console.log(texto.replace("mundo", "JavaScript")); // "Hola, JavaScript"

7. Dividir Texto ✂️​

🔀 split​

Divide una cadena en un array usando un delimitador.

Ejemplo:

let texto = "JavaScript,HTML,CSS";
console.log(texto.split(",")); // ["JavaScript", "HTML", "CSS"]

8. Juntar Texto dividio 🧩​

🔗 join​

Combina los elementos de un array en una cadena.

Ejemplo:

let texto = ["Lenguajes", "de", "Marcas"]
console.log(texto.join(" ")); // "Lenguajes de Marcas"]

9. Eliminar Espacios 🧹​

🚿 trim​

Elimina espacios en blanco al inicio y al final de una cadena.

Ejemplo:

let texto = "   Hola, mundo   ";
console.log(texto.trim()); // "Hola, mundo"

10. Repetir Texto 🔁​

🔄 repeat​

Devuelve una nueva cadena que repite el texto un número específico de veces.

Ejemplo:

let texto = "Hola ";
console.log(texto.repeat(3)); // "Hola Hola Hola "

Buenas Prácticas 👍🏼​

sigue estos consejos...

✅ 1. Usa Template Literals para Concatenación Compleja:

  • Más legible y fácil de mantener. 🛠️
let nombre = "Juan";
let edad = 25;
console.log(`Hola, me llamo ${nombre} y tengo ${edad} años.`);

✅ 2. Evita Hardcoding de Textos:

  • Usa variables para almacenar textos que se repiten. 💾

✅ 3. Valida las Entradas del Usuario:

  • Usa métodos como trim y toLowerCase para normalizar los datos ingresados. ✅
