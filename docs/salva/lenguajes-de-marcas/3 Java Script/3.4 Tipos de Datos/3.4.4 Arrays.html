3.4.4 Arrays

Los arrays en JavaScript son 🗃️ estructuras de datos que permiten almacenar múltiples valores en una sola variable. Son dinámicos ✨, lo que significa que pueden crecer o decrecer según sea necesario, y pueden contener cualquier tipo de dato (números 🔢, cadenas 📝, objetos 🛠️, funciones ➡️, otros arrays 🔄, etc.).

¿Qué es un Array?​

Un array es una colección ordenada de elementos 🗂️, donde cada elemento tiene un índice numérico que comienza en 0.

Ejemplo Básico 💡​

let numeros = [1, 2, 3, 4, 5];
console.log(numeros[0]); // 1
console.log(numeros[3]); // 4

Cómo Crear un Array​

1. Declaración usando Corchetes 🖇️​

Es la forma más común de declarar un array.

let frutas = ["manzana", "naranja", "plátano"];

2. Declaración usando Array 🏗️​

let numeros = new Array(1, 2, 3, 4, 5);

3. Array Vacío 🕳️​

let vacio = [];

4. Array con Longitud Específica 📏​

Crea un array con un tamaño definido (pero vacío).

let longitudEspecifica = new Array(5);
console.log(longitudEspecifica.length); // 5

Propiedades de los Arrays​

1. length 🔢​

Devuelve el número de elementos en un array.

let frutas = ["manzana", "naranja", "plátano"];
console.log(frutas.length); // 3

Métodos Comunes de los Arrays​

1. Agregar ➕ y Eliminar ➖ Elementos​

🔽 push​

Agrega uno o más elementos al final del array.

let numeros = [1, 2, 3];
numeros.push(4);
console.log(numeros); // [1, 2, 3, 4]

🔼 pop​

Elimina y devuelve el último elemento del array.

let numeros = [1, 2, 3];
let eliminado = numeros.pop();
console.log(eliminado); // 3
console.log(numeros); // [1, 2]

🔝 unshift​

Agrega uno o más elementos al principio del array.

let numeros = [2, 3];
numeros.unshift(1);
console.log(numeros); // [1, 2, 3]

🔝 shift​

Elimina y devuelve el primer elemento del array.

let numeros = [1, 2, 3];
let eliminado = numeros.shift();
console.log(eliminado); // 1
console.log(numeros); // [2, 3]

2. Ordenar y Revertir 🔄​

📜 sort​

Ordena los elementos del array alfabéticamente por defecto.

let numeros = [10, 2, 5, 1];
numeros.sort();
console.log(numeros); // [1, 10, 2, 5]

↩️ reverse​

Invierte el orden de los elementos en el array.

let numeros = [1, 2, 3];
numeros.reverse();
console.log(numeros); // [3, 2, 1]

3. Buscar Elementos 🔍​

📍 indexOf​

Devuelve el índice de la primera aparición de un elemento.

let frutas = ["manzana", "naranja", "plátano"];
console.log(frutas.indexOf("naranja")); // 1

✅ includes​

Devuelve true si el elemento está presente en el array.

let frutas = ["manzana", "naranja", "plátano"];
console.log(frutas.includes("naranja")); // true

4. Métodos Funcionales 🔁​

🔄 forEach​

Ejecuta una función para cada elemento del array.

let numeros = [1, 2, 3];
numeros.forEach((numero) => {
    console.log(numero * 2);
});

🚀 map​

Crea un nuevo array aplicando una función a cada elemento.

let numeros = [1, 2, 3];
let duplicados = numeros.map((numero) => numero * 2);
console.log(duplicados); // [2, 4, 6]

🔍 filter​

Crea un nuevo array con los elementos que cumplen una condición.

let numeros = [1, 2, 3, 4];
let pares = numeros.filter((numero) => numero % 2 === 0);
console.log(pares); // [2, 4]

📉 reduce​

Reduce los elementos del array a un único valor usando una función acumuladora.

let numeros = [1, 2, 3, 4];
let suma = numeros.reduce((acumulador, numero) => acumulador + numero, 0);
console.log(suma); // 10

5. Combinar 🔗 y Dividir ✂️​

➕ concat​

Combina dos o más arrays en uno nuevo.

let array1 = [1, 2];
let array2 = [3, 4];
let combinado = array1.concat(array2);
console.log(combinado); // [1, 2, 3, 4]

✂️ slice​

Crea un nuevo array copiando elementos desde un índice inicial hasta un índice final (sin incluir este último).

let numeros = [1, 2, 3, 4, 5];
console.log(numeros.slice(1, 4)); // [2, 3, 4]

🔄 splice​

Agrega, elimina o reemplaza elementos en un array.

Eliminar elementos:

let numeros = [1, 2, 3, 4];
numeros.splice(1, 2); // Elimina 2 elementos desde el índice 1
console.log(numeros); // [1, 4]

Agregar elementos:

numeros.splice(1, 0, 2, 3); // Agrega 2 y 3 en el índice 1
console.log(numeros); // [1, 2, 3, 4]

Arrays Multidimensionales​

Un array multidimensional es un array que contiene otros arrays como elementos 🗂️.

Ejemplo:

let matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

console.log(matriz[1][2]); // 6

Buenas Prácticas 👍🏼​

sigue estos consejos...

✔️ Usa métodos funcionales cuando sea posible

  • Métodos como map, filter y reduce son más legibles y eficientes.

✔️ Evita modificar arrays originales

  • Métodos como slice crean copias, lo cual es útil para evitar efectos secundarios.

✔️ Declara arrays con const si no necesitas reasignarlos

  • Esto ayuda a evitar errores al modificar referencias.

✔️ Valida índices antes de acceder

  • Evita errores asegurándote de que el índice está dentro del rango válido.
