3.1. Introducción a JavaScript

¿Qué es JavaScript? 🌐​

JavaScript es un lenguaje de programación dinámico, interpretado y de alto nivel, diseñado inicialmente para ejecutarse en los navegadores web 🌍. Fue creado para añadir interactividad y dinamismo a las páginas web y hoy en día es un pilar fundamental en el desarrollo web moderno. 🚀

Características principales de JavaScript 🔑​

  1. Lenguaje Interpretado 📝:

    • No requiere una fase de compilación previa ⚙️.
    • El código se ejecuta directamente en el entorno del navegador 🌟.

  2. Multiparadigma 🧩:

    • Permite programar en estilos imperativo, orientado a objetos y funcional. 🎭

  3. Asíncrono y No Bloqueante ⏱️:

    • Con características como eventos, promesas y async/await, permite manejar tareas sin detener el flujo del programa. 🔄

  4. Manipulación del DOM 🏗️:

    • Permite interactuar y modificar dinámicamente el contenido de una página web. 📜

  5. Portabilidad 🖥️:

    • Es compatible con todos los navegadores modernos. 🌈

Historia de JavaScript 📜​

  • 1995: Creación de JavaScript por Brendan Eich en solo 10 días para el navegador Netscape Navigator. Inicialmente, se llamó Mocha ☕ y luego LiveScript antes de adoptar el nombre JavaScript.

  • 1996: Microsoft desarrolla JScript, una versión propia de JavaScript, integrada en Internet Explorer. 🖥️

  • 1997: JavaScript es estandarizado por ECMA bajo el nombre ECMAScript (ES). 🛠️

  • Actualidad: Con la llegada de ES6 (ECMAScript 2015) y versiones posteriores, el lenguaje se ha enriquecido significativamente con nuevas características como let, const, clases, módulos, promesas, y más. 🌟

¿Dónde se ejecuta JavaScript? 🌎​

1. Navegadores Web 🌐​

La ejecución principal de JavaScript ocurre en los navegadores del lado del cliente. Algunos motores JavaScript populares incluyen:

  • V8: Utilizado por Chrome y Node.js. ⚡
  • SpiderMonkey: Utilizado por Mozilla Firefox. 🦊
  • JavaScriptCore: Utilizado por Safari. 🍏

Ejemplo básico: Un script que imprime "Hola, mundo" en la consola del navegador.

<!DOCTYPE html>
<html>
<head>
    <title>Ejemplo JavaScript</title>
</head>
<body>
    <script>
        console.log("Hola, mundo");
    </script>
</body>
</html>

2. Servidores (Node.js) 🖥️​

Con Node.js, un entorno basado en el motor V8, JavaScript se utiliza para el desarrollo del lado del servidor. Permite manejar solicitudes HTTP, conectarse a bases de datos, y más. 🌐

¿Por qué es tan popular JavaScript? ⭐​

  1. Versatilidad 🛠️:

    • JavaScript se utiliza tanto en el front-end como en el back-end, así como en aplicaciones móviles 📱, de escritorio 🖥️, y hasta para IoT. 🌟

  2. Facilidad de Aprendizaje 📚:

    • Tiene una sintaxis sencilla, ideal para principiantes. 👶

  3. Ecosistema Amplio 🌳:

    • La comunidad es enorme, con bibliotecas populares como React, Vue, Angular, y un gestor de paquetes extensivo (npm). 📦

  4. Integración Directa con HTML y CSS 🖌️:

    • JavaScript trabaja de manera conjunta con HTML (estructura) y CSS (diseño) para crear experiencias de usuario completas. 🎨

Diferencias entre JavaScript y Java 🤔​

AspectoJavaScriptJava
TipoInterpretado, dinámicoCompilado, estático
EjecuciónNavegadores y servidoresMáquina virtual Java (JVM)
SintaxisMás flexibleMás estricta
Uso PrincipalInteractividad webAplicaciones empresariales y móviles

Ejemplo Práctico: Introducción Dinámica al Usuario 🛠️​

HTML​

<!DOCTYPE html>
<html>
<head>
    <title>Ejemplo Interactivo</title>
</head>
<body>
    <h1 id="saludo">Bienvenido</h1>
    <button id="boton">Haz clic aquí</button>

    <script src="script.js"></script>
</body>
</html>

JavaScript (script.js)s​

// Selección de elementos
let saludo = document.getElementById("saludo");
let boton = document.getElementById("boton");

// Evento en el botón
boton.addEventListener("click", () => {
    let nombre = prompt("¿Cómo te llamas?");
    if (nombre) {
        saludo.textContent = `¡Hola, ${nombre}! Bienvenido a JavaScript`;
    } else {
        saludo.textContent = "¡Hola, visitante!";
    }
});

Resultado: Al hacer clic en el botón, el usuario puede introducir su nombre, y el mensaje se actualiza dinámicamente. 🎉
