Configuración del Proyecto

En este primer paso, configuraremos el entorno de desarrollo de nuestra aplicación en Android Studio. Esto incluye crear el proyecto, habilitar ViewBinding, agregar las dependencias necesarias y preparar todo para empezar a trabajar con Room, MVVM y RecyclerView.

Crea un nuevo proyecto​

Para comenzar crearemos un nuevo proyecto con las siguientes características...

Habilita ViewBinding​

ViewBinding nos permite acceder a las vistas del diseño XML de manera más sencilla y segura, evitando errores comunes como los "NullPointerException".

  1. Abre el archivo build.gradle del módulo

  2. Habilita ViewBinding

    build.gradle.kts (Module :app)
      android {
          ...
          viewBinding {
              enabled = true
          }
      }

Agregar dependencias necesarias​

Las dependencias son bibliotecas externas que nos permiten añadir funcionalidades avanzadas a nuestra aplicación, como la gestión de bases de datos con Room o la visualización de listas con RecyclerView.

  1. Abre el archivo build.gradle del módulo

  2. Añade las dependencias

    • En el bloque dependencies { ... }, copia y pega lo siguiente:
      build.gradle.kts (Module :app)
          dependencies {
            // Room
             implementation ("androidx.room:room-runtime:2.6.1")
             annotationProcessor ("androidx.room:room-compiler:2.6.1")

           // LiveData y ViewModel
             implementation ("androidx.lifecycle:lifecycle-livedata-ktx:2.8.7")
             implementation ("androidx.lifecycle:lifecycle-viewmodel-ktx:2.8.7")

           // RecyclerView
             implementation ("androidx.recyclerview:recyclerview:1.3.2")
          }
info
  • Room: Es la biblioteca oficial de Android para trabajar con bases de datos SQLite.
  • LiveData y ViewModel: Nos ayudan a implementar la arquitectura MVVM, separando la lógica de negocio de la interfaz de usuario.

Sincronizar el proyecto​

Recuerda...
  1. Haz clic en "Sync Now" Si ves un error, verifica que hayas copiado correctamente las dependencias.

¡Felicidades! 🎉​

Has configurado correctamente el proyecto. Ahora tenemos el entorno preparado para empezar a trabajar en nuestra aplicación.

  • Habilita ViewBinding
  • Agregar dependencias necesarias
  • Sincronizar el proyecto
