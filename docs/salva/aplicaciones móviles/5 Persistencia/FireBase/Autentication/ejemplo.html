Ejemplo Email-Password

A continuación, describimos los pasos para crear una app muy básica donde el usuario pueda registrarse e iniciar sesión con correo y contraseña usando Firebase Authentication. La aplicación contendrá 2 Activities, una donde podremos logarnos o registrarnos y la otra nos dará la bienvenida indicando si el usuario es nuevo o antiguo.

1. Crear el proyecto en Android Studio​

  • Selecciona File > New > New Project.
  • Escoge una plantilla vacía (Empty Activity).
  • Configura el nombre del proyecto y el paquete.
API LEVEL

La versión mínima (minSdkVersion) recomendada para utilizar la mayoría de las bibliotecas de Firebase en Android suele ser API 19 (Android 4.4 – KitKat).

Sin embargo, no todas las bibliotecas de Firebase tienen exactamente el mismo requisito, ya que algunas pueden funcionar con una API inferior (por ejemplo, API 16) y otras pueden requerir API 19 o superior. En general, si usas la última versión de las bibliotecas de Firebase, lo más seguro es establecer minSdkVersion en 28 para evitar incompatibilidades.

2. Configurar Firebase​

  • Sigue los pasos de la sección configuración (descargar google-services.json, habilitar Email/Password en la consola de Firebase, etc.).

4. Programar las Activities​

Actividad Registro​

Esta será la actividad que permita al usuario hacer LogIn o Registrarse. Crea la clase RegistroActivity.java y escribe el siguiente código:

RegistroActivity.java
package com.agora.autenticaciondemo;

import androidx.annotation.NonNull;
import androidx.appcompat.app.AppCompatActivity;

import android.content.Intent;
import android.os.Bundle;
import android.view.View;
import android.widget.Toast;

import com.agora.autenticaciondemo.databinding.ActivityRegistroBinding;
import com.google.android.gms.tasks.OnCompleteListener;
import com.google.android.gms.tasks.Task;
import com.google.firebase.FirebaseApp;
import com.google.firebase.auth.AuthResult;
import com.google.firebase.auth.FirebaseAuth;
import com.google.firebase.auth.FirebaseUser;

/**
* RegisterActivity maneja el registro e inicio de sesión con Firebase Authentication.
*/
public class RegistroActivity extends AppCompatActivity {

 private ActivityRegistroBinding binding;
 private FirebaseAuth mAuth;

 @Override
 protected void onCreate(Bundle savedInstanceState) {
     super.onCreate(savedInstanceState);
     // Inicializamos View Binding
     binding = ActivityRegistroBinding.inflate(getLayoutInflater());
     setContentView(binding.getRoot());

     // Inicializar Firebase
     FirebaseApp.initializeApp(this);
     mAuth = FirebaseAuth.getInstance();

     // Click en "Iniciar Sesión"
     binding.btnLogin.setOnClickListener(new View.OnClickListener() {
         @Override
         public void onClick(View view) {
             logarUsuario();
         }
     });

     // Click en "Registrarse"
     binding.btnRegistro.setOnClickListener(new View.OnClickListener() {
         @Override
         public void onClick(View view) {
             crearUsuario();
         }
     });
 }

 /**
  * Crea un nuevo usuario en Firebase (Email/Password).
  */
 private void crearUsuario() {
     String email = binding.textoMail.getText().toString().trim();
     String password = binding.textoPassword.getText().toString().trim();
     String password2 = binding.textoPassword2.getText().toString().trim();

     if (email.isEmpty() || password.isEmpty() || password2.isEmpty()) {
         Toast.makeText(this, "Por favor, completa todos los campos", Toast.LENGTH_SHORT).show();
         return;
     } else if (!password.equals(password2)) {
         Toast.makeText(this, "Las contraseñas no coinciden", Toast.LENGTH_SHORT).show();
         return;
     }

     mAuth.createUserWithEmailAndPassword(email, password)
             .addOnCompleteListener(this, new OnCompleteListener<AuthResult>() {
                 @Override
                 public void onComplete(@NonNull Task<AuthResult> task) {
                     if (task.isSuccessful()) {
                         // Usuario creado por primera vez
                         FirebaseUser usuario = mAuth.getCurrentUser();
                         if (usuario != null) {
                             Toast.makeText(RegistroActivity.this,
                                     "Bienvenido, nuevo usuario: " + usuario.getEmail(),
                                     Toast.LENGTH_SHORT).show();
                         }
                         // Pasamos a la actividad principal
                         iniciarMainActivity();
                     } else {
                         // Falla en la creación de usuario (e.g., email ya en uso)
                         Toast.makeText(RegistroActivity.this,
                                 "Error al crear usuario: " + task.getException().getMessage(),
                                 Toast.LENGTH_SHORT).show();
                     }
                 }
             });
 }

 /**
  * Inicia sesión con un usuario existente.
  */
 private void logarUsuario() {
     String email = binding.textoMail.getText().toString().trim();
     String password = binding.textoPassword.getText().toString().trim();

     if (email.isEmpty() || password.isEmpty()) {
         Toast.makeText(this, "Por favor, completa todos los campos", Toast.LENGTH_SHORT).show();
         return;
     }

     mAuth.signInWithEmailAndPassword(email, password)
             .addOnCompleteListener(this, new OnCompleteListener<AuthResult>() {
                 @Override
                 public void onComplete(@NonNull Task<AuthResult> task) {
                     if (task.isSuccessful()) {
                         // Usuario existente
                         FirebaseUser usuario = mAuth.getCurrentUser();
                         if (usuario != null) {
                             Toast.makeText(RegistroActivity.this,
                                     "Bienvenido de nuevo: " + usuario.getEmail(),
                                     Toast.LENGTH_SHORT).show();
                         }
                         // Pasamos a la actividad principal
                         iniciarMainActivity();
                     } else {
                         // Error al iniciar sesión
                         Toast.makeText(RegistroActivity.this,
                                 "Error al iniciar sesión: " + task.getException().getMessage(),
                                 Toast.LENGTH_SHORT).show();
                     }
                 }
             });
 }

 /**
  * Navega a la MainActivity
  */
 private void iniciarMainActivity() {
     Intent intent = new Intent(RegistroActivity.this, MainActivity.class);
     startActivity(intent);
     finish();
 }
}

create User With Email And Password

Esta función es el proceso de registro de un usuario con Firebase Authentication empleando correo y contraseña. Este método es parte de Firebase Authentication y permite, de forma rápida, implementar registro en tu aplicación Android sin tener que crear tu propio servidor de autenticación ni manejar contraseñas en tu backend. Veámosla paso a paso:

  1. mAuth.createUserWithEmailAndPassword(email, password)

    • Llama al método de FirebaseAuth para crear un usuario nuevo usando correo y contraseña.
    • Devuelve un objeto de tipo Task<AuthResult> que representa la operación asíncrona (ocurre en segundo plano).

  2. addOnCompleteListener(this, new OnCompleteListener<AuthResult>() {...})

    • Se registra un listener que se invoca cuando la operación termine (se complete).
    • El primer parámetro (this) indica el Context o LifecycleOwner que escucha el resultado (en este caso, la Activity actual).
    • El segundo parámetro es el código que se ejecutará una vez que la tarea (registro) finalice (exitosamente o con error).

  3. onComplete(@NonNull Task<AuthResult> task)

    • Este método se dispara al terminar la creación del usuario.
    • El objeto task contiene información del resultado (si fue exitoso o no).

  4. if (task.isSuccessful()) { ... } else { ... }

    • Se verifica si la operación fue exitosa o falló.
    • task.isSuccessful() retornará true si el registro se efectuó correctamente.

  5. Cuando es exitoso

    • Se obtiene el usuario recién creado con mAuth.getCurrentUser().
    • Si no es null, se muestra un mensaje de bienvenida que incluye su email.
    • Después, se llama a iniciarMainActivity() para cambiar a la siguiente pantalla (por ejemplo, la pantalla principal de la app).

  6. Cuando falla

    • Se muestra un mensaje de error que incluye la descripción de la excepción: task.getException().getMessage().
    • Una de las causas comunes de error podría ser que el email ya esté en uso o que no cumpla con el formato requerido por Firebase.
sign In With Email And Password

Este método es clave para iniciar sesión con Firebase Authentication, permitiendo a los usuarios que ya se registraron acceder a tu aplicación de forma segura.

  1. mAuth.signInWithEmailAndPassword(email, password)

    • Este método de FirebaseAuth realiza la autenticación de un usuario existente con las credenciales suministradas (email y contraseña).
    • Regresa un objeto Task<AuthResult> para manejar la operación de manera asíncrona (ocurre en segundo plano).

  2. addOnCompleteListener(this, new OnCompleteListener<AuthResult>() { ... })

    • Se registra un listener que se ejecutará al finalizar la operación (ya sea con éxito o error).
    • El primer parámetro (this) indica el Context de la Activity que escucha la respuesta.
    • Dentro del listener se define el comportamiento en onComplete.

  3. onComplete(@NonNull Task<AuthResult> task)

    • Se invoca cuando la tarea concluye.
    • El objeto task contiene la información sobre el resultado de la autenticación (éxito o fallo).

  4. if (task.isSuccessful()) { ... } else { ... }

    • Se pregunta si la autenticación fue exitosa.
    • task.isSuccessful() retorna true cuando el login con email y contraseña se realiza correctamente.

  5. Si la autenticación es exitosa

    • Se obtiene el usuario autenticado con mAuth.getCurrentUser().
    • Si no es null, se muestra un mensaje personalizado: “Bienvenido de nuevo: …”.
    • Después, se llama a iniciarMainActivity() para redirigir a la siguiente pantalla (por ejemplo, la principal de la app).

  6. Si la autenticación falla

    • Se muestra un Toast con el mensaje de error: task.getException().getMessage().
    • Algunas razones comunes pueden ser credenciales inválidas (contraseña incorrecta), email no registrado, etc.

Actividad Principal​

Crea la clase MainActivity.java para mostrar la información del usuario y un botón de logout. También cerraremos sesión automáticamente al salir de la app.

MainActivity.java
package com.agora.autenticaciondemo;

import androidx.appcompat.app.AppCompatActivity;

import android.content.Intent;
import android.os.Bundle;
import android.util.Log;
import android.view.View;
import android.widget.Toast;

import com.agora.autenticaciondemo.databinding.ActivityMainBinding;
import com.google.firebase.auth.FirebaseAuth;
import com.google.firebase.auth.FirebaseUser;
import com.google.firebase.auth.FirebaseUserMetadata;

import java.util.Date;

/**
 * MainActivity muestra información del usuario y permite cerrar sesión.
 */
public class MainActivity extends AppCompatActivity {

    // View Binding para activity_main.xml
    private ActivityMainBinding binding;

    // Referencia a FirebaseAuth
    private FirebaseAuth mAuth;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);

        // Inicializamos el binding
        binding = ActivityMainBinding.inflate(getLayoutInflater());
        View view = binding.getRoot();
        setContentView(view);

        // Obtener instancia de FirebaseAuth
        mAuth = FirebaseAuth.getInstance();

        // Verificar si hay un usuario logueado
        FirebaseUser usuarioActual = mAuth.getCurrentUser();
        if (usuarioActual == null) {
            // Si no hay usuario, volvemos a RegistroActivity
            volverRegistro();
        } else {
            //Mostramos email del usuario
            binding.textViewUserInfo.setText(usuarioActual.getEmail());
            // Obtener la metadata del usuario
            FirebaseUserMetadata metadata = usuarioActual.getMetadata();
            if (metadata != null) {
                // Obtener la fecha de creación y último inicio de sesión
                    long creationTime = metadata.getCreationTimestamp();      // Milisegundos desde la época Unix
                    long lastSignInTime = metadata.getLastSignInTimestamp();  // Milisegundos desde la época Unix
                    // Para mostrarlo de forma legible, podemos convertirlo a Date:
                    Date creationDate = new Date(creationTime);
                    Date lastSignInDate = new Date(lastSignInTime);
                //Mostramos información del usuario
                    binding.textViewUserInfo2.setText("Cuenta creada: " + creationDate.toString());
                    binding.textViewUserInfo3.setText("Última sesión: " + lastSignInDate.toString());
              }
        }

        // Evento para botón "Cerrar Sesión"
        binding.buttonLogout.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View view) {
                signOutUsuario();
            }
        });
    }

    /**
     * Cierra la sesión del usuario.
     */
    private void signOutUsuario() {
        mAuth.signOut();
        Toast.makeText(MainActivity.this, "Sesión cerrada", Toast.LENGTH_SHORT).show();

        // Volver a la pantalla de registro/inicio
        volverRegistro();
    }

    /**
     * Navegar a RegisterActivity.
     */
    private void volverRegistro() {
        Intent intent = new Intent(MainActivity.this, RegistroActivity.class);
        startActivity(intent);
        finish(); // Cerramos la actividad actual
    }

    /**
     * Cuando el usuario sale de la app (onDestroy se llama cuando la Activity se destruye),
     * cerramos sesión automáticamente según la solicitud.
     */
    @Override
    protected void onDestroy() {
        super.onDestroy();
        // Cerrar sesión automáticamente al destruir la Activity (salir de la app)
        mAuth.signOut();
    }
}


get metadata

El método getMetadata() se llama sobre un objeto de tipo FirebaseUser y devuelve un objeto de tipo FirebaseUserMetadata. El objeto FirebaseUserMetadata proporciona dos valores principales relacionados con la cuenta del usuario:

  • Creation Timestamp (getCreationTimestamp()): Es un valor en milisegundos que indica cuándo se creó la cuenta por primera vez en Firebase (momento del registro).
  • Last Sign-In Timestamp (getLastSignInTimestamp()):También es un valor en milisegundos que indica cuándo fue la última vez que el usuario inició sesión exitosamente.
onDestroy vs. onStop

Aquí usamos onDestroy() para cerrar la sesión cuando la actividad principal se destruye. Dependiendo del ciclo de vida y del dispositivo (por ejemplo, rotaciones de pantalla), onDestroy() puede ser llamado en otras circunstancias. Si prefieres, podrías implementar este cierre de sesión en onStop() o manejarlo según la lógica de tu aplicación para evitar efectos inesperados (por ejemplo, si el usuario gira la pantalla se recrea la actividad y volvería a RegisterActivity).

5. Modifica el AndroidManifest​

Para que la actividad que arranque la aplicación sea la RegistroActivity deberemos modificar el archivo AndroidManifest.xml. Debe quedarte algo similar a esto.

AndroidManifest.xml
<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools">

    <application
        android:allowBackup="true"
        android:dataExtractionRules="@xml/data_extraction_rules"
        android:fullBackupContent="@xml/backup_rules"
        android:icon="@mipmap/ic_launcher"
        android:label="@string/app_name"
        android:roundIcon="@mipmap/ic_launcher_round"
        android:supportsRtl="true"
        android:theme="@style/Theme.AutenticacionDemo"
        tools:targetApi="31">
        <activity
            android:name=".RegistroActivity"
            android:exported="true">
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />
                <category android:name="android.intent.category.LAUNCHER" />
            </intent-filter>
        </activity>
        <activity
            android:name=".MainActivity"
            android:exported="true">
        </activity>
    </application>

</manifest>
Intent-filter

Asegúrate que la etiqueta <intent-filter> con el action MAIN y category LAUNCHER este dentro de la actividad de registro.
