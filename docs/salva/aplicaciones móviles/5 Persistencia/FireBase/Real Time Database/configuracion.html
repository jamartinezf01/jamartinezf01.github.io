Configuración

Para configurar Firebase Realtime Database en tu proyecto de Firebase, sigue estos pasos:

1. Crea o selecciona un proyecto de Firebase​

  • Ve a la Consola de Firebase, inicia sesión con tu cuenta de Google y crea un proyecto nuevo (o selecciona uno existente).

2. Habilita el producto de RealTime Database y crea una nueva base de datos​

  • En la barra lateral, selecciona Compilación > RealtimeDatabase.
  • Pulsa Crear una base de datos

3. Configura los prámetros necesarios​

  • Selecciona la ubicación más cercana a tus clientes. En nuestro caso Bélgica (europe-west1).

  • Selección las reglas de seguridad. Es recomendable que para el desarrollo de tu aplicación uses el modo prueba.

info

Para saber más sobre las reglas de seguridad visita la Documentación de Firebase

4. Agrega las dependencias​

En el archivo build.gradle.kts del módulo (nivel de la app), agrega la dependencia del módulo de la base de datos:

<project>/<app-module>/build.gradle.kts

dependencies {
    // Otras dependencias
    
    // Dependencia para Realtime Database
    implementation("com.google.firebase:firebase-database:21.0.0")

    // ...
}

5. Descargar y actualizar google-services.json​

Recuerda que cada vez que agreges un servicio deberas...

  • Desdescargar el archivo de configuración google-services.json.
  • Colocar este archivo dentro de la carpeta app/ de tu proyecto Android.

6. ¡Felicidades! 🎉​

Una vez realizado esto, tu proyecto estará listo para usar Firebase Realtime Database.

  • 1. Crea o selecciona un proyecto de Firebase
  • 2. Habilita el producto de RealTime Database y crea una nueva base de datos
  • 3. Configura los prámetros necesarios
  • 4. Agrega las dependencias
  • 5. Descargar y actualizar google-services.json
  • 6. ¡Felicidades! 🎉
